🧠 THE PROBLEM: THE ELIMINATION WALL

In Chapter 6, you learned the elimination method: three answers are WRONG before one is RIGHT. You learned to identify distractors by checking them against the text and removing those with no support.

That method works perfectly — until it doesn't.

Here is what happens at a higher difficulty level:

THE ELIMINATION WALL:
═══════════════════════════════════════════════════════════

  EASY QUESTION (Chapters 6-8 level):
  ┌────────────────────────────────────────┐
  │  Option A → Obviously wrong (no text   │ → ELIMINATE
  │              support at all)            │
  │  Option B → Obviously wrong (contradicts│ → ELIMINATE
  │              the text directly)          │
  │  Option C → Clearly correct (matches    │ → SELECT ✓
  │              text precisely)             │
  │  Option D → Obviously wrong (invented   │ → ELIMINATE
  │              information)                │
  │                                          │
  │  DIFFICULTY: 1 correct + 3 clearly wrong │
  │  RESULT: Easy selection                  │
  └────────────────────────────────────────┘

  HARD QUESTION (Chapter 18 level):
  ┌────────────────────────────────────────┐
  │  Option A → Seems wrong               │ → ELIMINATE
  │  Option B → Has SOME text support      │ → KEEP ⚠️
  │  Option C → Has STRONG text support    │ → KEEP ⚠️
  │  Option D → Seems wrong               │ → ELIMINATE
  │                                          │

  │  └──────────────────────────────┘        │
  └────────────────────────────────────────┘

═══════════════════════════════════════════════════════════

This is the elimination wall. Standard elimination gets you from 4 options to 2. But the final step — choosing between two plausible answers — requires a different tool entirely.

📌 Elimination is a REDUCTION tool. It narrows the field. But it cannot make the FINAL selection when two options survive. For that, you need a SCORING tool.

🔍 HOW TO APPROACH IT: UNDERSTANDING WHY TWO ANSWERS SEEM RIGHT

Before building the scoring tool, you must understand the four reasons why two answers can both appear correct:

THE FOUR TYPES OF TIE SITUATIONS:
═══════════════════════════════════════════════════════════

  TYPE 1: "PARTIALLY TRUE vs COMPLETELY TRUE"
  ──────────────────────────────────────────
  One answer captures PART of the text's message.
  The other captures ALL of it.
  Both have textual support, but one is MORE COMPLETE.
  
  TYPE 2: "TRUE BUT WRONG QUESTION"
  ──────────────────────────────────────────
  One answer is a TRUE statement about the text
  but does NOT answer the SPECIFIC question asked.
  The other answers the question directly.
  (Chapter 19 will explore this in depth)
  
  TYPE 3: "CLOSE PARAPHRASE vs EXACT PARAPHRASE"
  ──────────────────────────────────────────
  Both answers rephrase the text, but one introduces
  a subtle DISTORTION (added word, shifted emphasis,
  slight exaggeration) while the other stays faithful.
  
  TYPE 4: "GENERAL vs SPECIFIC"
  ──────────────────────────────────────────
  One answer is broadly correct but vague.
  The other is precisely correct and specific.
  The specific one is almost always better.

═══════════════════════════════════════════════════════════

Let us see each type in action with real questions from our texts.

🔧 HOW CAN WE SOLVE IT: THE DEGREE-OF-FIT SCORING SYSTEM

THE CORE PRINCIPLE

When two answers both seem right, score each one against the text using five criteria. The answer with the higher total score wins.

THE FIVE SCORING CRITERIA:
═══════════════════════════════════════════════════════════

  CRITERION 1: TEXTUAL PROOF (0-3 points)
  → How many specific lines in the text DIRECTLY support 
    this answer?
  → 0 = no direct support
  → 1 = one line partially supports it
  → 2 = one line clearly supports it
  → 3 = multiple lines clearly support it

  CRITERION 2: COMPLETENESS (0-2 points)
  → Does this answer capture the FULL scope of what the 
    question asks, or only PART of it?
  → 0 = captures only a minor aspect
  → 1 = captures the main point but misses nuance
  → 2 = captures the complete picture

  CRITERION 3: PRECISION (0-2 points)
  → Does this answer use words that MATCH the text's 
    actual vocabulary and meaning, or does it introduce
    terms/concepts not in the text?
  → 0 = introduces new concepts not in the text
  → 1 = close but slightly shifted meaning
  → 2 = precise match with text vocabulary

  CRITERION 4: NO ADDED INFORMATION (0-2 points)
  → Does this answer stay within what the text says,
    or does it ADD something the text never states?
  → 0 = adds significant information not in text
  → 1 = slightly extends beyond text
  → 2 = stays strictly within text boundaries

  CRITERION 5: QUESTION ALIGNMENT (0-1 point)
  → Does this answer address the EXACT question asked,
    or does it answer a slightly different question?
  → 0 = answers a different question
  → 1 = answers the exact question

  MAXIMUM SCORE: 3 + 2 + 2 + 2 + 1 = 10 points
  
═══════════════════════════════════════════════════════════

TIE SITUATION 1: TEXTE 101, QUESTION 24 (Type 1 — Partially True vs Completely True)

Question 24: cet article a pour but
 A. de dénoncer les thèses avant-gardistes publiées par un chercheur.
 B. de convaincre des avantages de la future révolution technologique.
 C. d'informer des évolutions technologiques avancées par un scientifique.
 D. de présenter des réalisations révolutionnaires d'une vedette de la science.

Chapter 6 covered the elimination of A and B. Chapter 2 identified the genre as informative journalism. But what about C vs D? Both sound plausible:

FIRST PASS ELIMINATION (Chapter 6 method):
                                                          
  A. "dénoncer" (denounce) → The text doesn't condemn Kurzweil
     → Neutral tone throughout → ELIMINATE
     
  B. "convaincre" (convince) → The text doesn't argue FOR
     the revolution → It REPORTS on predictions → ELIMINATE
     
  C. "informer des évolutions technologiques avancées 
      par un scientifique" 
     → Inform about technological developments proposed 
       by a scientist → KEEP ⚠️
     
  D. "présenter des réalisations révolutionnaires d'une 
      vedette de la science"
     → Present revolutionary ACHIEVEMENTS of a science star
     → KEEP ⚠️

  RESULT: C and D both survive. Now what?

Apply the Degree-of-Fit Scoring:

SCORING: OPTION C — "d'informer des évolutions 
technologiques avancées par un scientifique"
═══════════════════════════════════════════════════════════

  CRITERION 1: TEXTUAL PROOF                    [3/3]
  → "Ray Kurzweil est une vedette de la prospective 
     scientifique" → he IS a scientific figure ✓
  → "Ses prédictions" → the text DESCRIBES his predictions ✓
  → "Selon ses estimations" → the text REPORTS his views ✓
  → Three lines directly support "informer" about 
    technological developments proposed by a scientist
  
  CRITERION 2: COMPLETENESS                     [2/2]
  → "informer" captures the article's NEUTRAL REPORTING tone ✓
  → "évolutions technologiques" covers all his predictions ✓
  → "avancées par un scientifique" correctly attributes 
    the ideas to Kurzweil ✓
  → Captures the full picture
  
  CRITERION 3: PRECISION                        [2/2]
  → "informer" matches the journalistic genre perfectly ✓
  → "évolutions technologiques" matches the content ✓
  → "un scientifique" matches "prospective scientifique" ✓
  → No vocabulary distortion
  
  CRITERION 4: NO ADDED INFORMATION             [2/2]
  → Everything in this answer is in the text ✓
  → Nothing is invented or extrapolated ✓
  
  CRITERION 5: QUESTION ALIGNMENT               [1/1]
  → Question asks "le but" (the purpose) ✓
  → "Informer" IS a purpose ✓
  → Directly answers "what is the article's purpose"

  TOTAL SCORE FOR C: 10/10
  
═══════════════════════════════════════════════════════════


SCORING: OPTION D — "de présenter des réalisations 
révolutionnaires d'une vedette de la science"
═══════════════════════════════════════════════════════════

  CRITERION 1: TEXTUAL PROOF                    [1/3]
  → "vedette de la science" → the text DOES call him 
    "vedette de la prospective scientifique" ✓
  → BUT: "réalisations" (achievements/accomplishments)?
    The text describes PREDICTIONS, not achievements
  → Predictions ≠ réalisations
  → Only one element has direct text support
  
  CRITERION 2: COMPLETENESS                     [1/2]
  → "présenter" is acceptable as a purpose ✓
  → But "réalisations" misrepresents the content
  → Kurzweil has NOT achieved these things — he PREDICTS them
  → Captures the person (vedette) but distorts the content
  
  CRITERION 3: PRECISION                        [0/2]
  → "réalisations révolutionnaires" = revolutionary 
    ACHIEVEMENTS → things already accomplished
  → The text uses: "prédictions," "estimations," 
    "imagine," conditional tense ("serions," "pourraient")
  → EVERYTHING in the text says these are FUTURE predictions,
    NOT past achievements
  → Critical vocabulary mismatch ✗
  
  CRITERION 4: NO ADDED INFORMATION             [0/2]
  → "réalisations" ADDS the idea that these things 
    have been DONE → the text never says this
  → It transforms predictions into accomplishments
  → Significant addition not in text ✗
  
  CRITERION 5: QUESTION ALIGNMENT               [1/1]
  → "présenter" IS a purpose ✓
  → Answers the question form correctly

  TOTAL SCORE FOR D: 3/10
  
═══════════════════════════════════════════════════════════
SCORECARD COMPARISON:
                                                          
  Option C: ██████████ 10/10
  Option D: ███░░░░░░░  3/10
  
  WINNER: C — by a MASSIVE margin
  
  THE KEY DIFFERENCE: One single word — "réalisations"
  → D would be correct if it said "prédictions" instead
  → But "réalisations" = achievements (done)
  → The text is about PREDICTIONS (future, uncertain)
  → This one word drops D's score from potential 8 to actual 3

📌 LESSON TYPE 1: When two answers seem right, one often contains a SINGLE WORD that changes everything. "Réalisations" vs "prédictions" is the difference between 10/10 and 3/10. The scoring system forces you to examine each word.

TIE SITUATION 2: TEXTE 101, QUESTION 26 (Type 3 — Close Paraphrase vs Exact Paraphrase)

Question 26: les changements majeurs prédits par Ray Kurzweil sont
 A. la vie éternelle et le bonheur de tous.
 B. l'intelligence artificielle et la dépollution.
 C. la technologie humanisée et l'anticipation de l'avenir.
 D. La robotisation du quotidien et la dictature des machines.
FIRST PASS ELIMINATION:
                                                          
  C. "la technologie humanisée et l'anticipation de l'avenir"
     → "Humanized technology" — the text talks about 
       humans and machines FUSING, which could be seen 
       as "humanizing" technology
     → "Anticipation de l'avenir" — Kurzweil IS anticipating 
       the future
     → KEEP ⚠️
     
  D. "La robotisation du quotidien et la dictature des machines"
     → "Robotization of daily life" — the text mentions 
       "minuscules robots" but NOT daily life robotization
     → "Dictature des machines" — the text says fusion, 
       NOT domination/dictatorship
     → The text says differences become "négligeables," 
       NOT that machines take over
     → ELIMINATE ✗
     
  A. "la vie éternelle et le bonheur de tous"
     → "Vie éternelle" — the text says "vivre éternellement 
       sans vieillir" → MATCH for eternal life ✓
     → "Bonheur de tous" (happiness for all) — NOWHERE 
       in the text → PARTIALLY supported
     → KEEP ⚠️
     
  B. "l'intelligence artificielle et la dépollution"
     → "Intelligence artificielle" — mentioned in paragraph 3 
       as a concept, and paragraph 2 describes enhanced 
       intelligence → SOME support
     → "Dépollution" — paragraph 1 says "minuscules robots 
       annuleront les effets de la pollution" → MATCH ✓
     → KEEP ⚠️

  RESULT: A, B, and C all survive! Three-way tie!
  D is eliminated. Now we need to score A vs B vs C.

This is a THREE-WAY tie — the hardest scenario. Apply scoring to all three:

SCORING: OPTION A — "la vie éternelle et le bonheur de tous"
═══════════════════════════════════════════════════════════

  CRITERION 1: TEXTUAL PROOF                    [1/3]
  → "Vivre éternellement sans vieillir" → supports 
    "vie éternelle" ✓
  → "bonheur de tous" → ZERO textual support ✗
    The text NEVER mentions happiness or well-being
  → Only half the answer has proof
  
  CRITERION 2: COMPLETENESS                     [0/2]
  → "Vie éternelle" is ONE prediction, not the 
    "changements majeurs" (plural, major changes)
  → Missing: robots, pollution, brain upload, fusion, 
    intelligence expansion
  → Captures only a fraction of Kurzweil's predictions
  
  CRITERION 3: PRECISION                        [1/2]
  → "Vie éternelle" is close to "vivre éternellement" ✓
  → "Bonheur de tous" is completely invented ✗
  
  CRITERION 4: NO ADDED INFORMATION             [0/2]
  → "Bonheur de tous" is ADDED — not in text at all ✗
  → Major fabrication
  
  CRITERION 5: QUESTION ALIGNMENT               [1/1]
  → Answers what "changements majeurs" are ✓

  TOTAL SCORE FOR A: 3/10

═══════════════════════════════════════════════════════════


SCORING: OPTION B — "l'intelligence artificielle et la dépollution"
═══════════════════════════════════════════════════════════

  CRITERION 1: TEXTUAL PROOF                    [3/3]
  → "intelligence artificielle" → paragraph 3 lists it 
    explicitly ✓
  → Also: "l'intelligence humaine, améliorée, pourrait 
    commencer à s'étendre dans l'univers" → enhanced 
    intelligence ✓
  → "dépollution" → "de minuscules robots annuleront

  → BOTH parts have direct textual proof
  
  CRITERION 2: COMPLETENESS                     [2/2]
  → "Intelligence artificielle" covers the technology/

  → "Dépollution" covers the environmental prediction

  → Together they span the text's two MAJOR predicted

  → Good coverage of "changements majeurs"
  
  CRITERION 3: PRECISION                        [2/2]
  → "Intelligence artificielle" appears VERBATIM in 
    paragraph 3 ✓
  → "Dépollution" is a precise noun form of "annuler 
    les effets de la pollution" ✓
  → Both terms match the text's vocabulary closely
  
  CRITERION 4: NO ADDED INFORMATION             [2/2]
  → Both concepts are IN the text ✓
  → Nothing fabricated or extrapolated ✓
  
  CRITERION 5: QUESTION ALIGNMENT               [1/1]
  → Question asks about "changements majeurs prédits" ✓
  → AI and depollution ARE major predicted changes ✓

  TOTAL SCORE FOR B: 10/10

═══════════════════════════════════════════════════════════


SCORING: OPTION C — "la technologie humanisée et

═══════════════════════════════════════════════════════════

  CRITERION 1: TEXTUAL PROOF                    [1/3]

  → "anticipation de l'avenir" — Kurzweil DOES anticipate

  → Vocabulary mismatch ✗
  
  CRITERION 4: NO ADDED INFORMATION             [0/2]
  → "Technologie humanisée" is an INTERPRETATION

  → "Anticipation de l'avenir" reframes the text's

  CRITERION 5: QUESTION ALIGNMENT               [0/1]

TIE SITUATION 3: TEXTE 101, QUESTION 27 (Type 4 — General vs Specific)

Question 27: la polémique suscitée par l'ensemble de ces prédictions
 A. mettra un terme aux controverses traditionnelles.
 B. ravive l'éternel conflit entre idéalistes et réalistes.
 C. renvoie à des questions scientifiques et métaphysiques.

     → "Dieu" = God → METAPHYSICAL ✓
     → "Intelligence artificielle" = SCIENTIFIC ✓
     → KEEP ⚠️

  → "Éternel" (eternal) is added — the text doesn't 
    say this is an eternal conflict ✗
  → Significant addition
  
  CRITERION 5: QUESTION ALIGNMENT               [1/1]
  → The question asks what the controversy "renvoie à"

  → B answers this in general terms ✓

TIE SITUATION 4: TEXTE 101, QUESTION 25 (Type 2 — True But Wrong Question)

Question 25: la principale théorie de Ray Kurzweil affirme que
 A. les hommes et les outils technologiques ne feront plus qu'un.
 B. La technologie dépossédera l'homme de certaines de ses qualités.
 C. Les outils technologiques seront supérieurs en intelligence aux hommes.
 D. L'humanité utilisera la technologie pour améliorer son patrimoine génétique.
FIRST PASS ELIMINATION:
                                                          
  B. "La technologie dépossédera l'homme de certaines 
      de ses qualités" (Technology will strip humans 
      of some qualities)
     → The text says technology will ENHANCE humans, 
       not strip them
     → "l'intelligence humaine, améliorée" = improved
     → CONTRADICTS the text → ELIMINATE ✗
     
  D. "L'humanité utilisera la technologie pour améliorer 
      son patrimoine génétique" (Humanity will use tech 
      to improve genetic heritage)
     → "Patrimoine génétique" appears in Texte 102 
       (salmon), NOT in Texte 101
     → Texte 101 mentions "génie génétique" as a 
       controversial THEME, not as something Kurzweil 
       specifically predicts humans will improve
     → WEAK support → ELIMINATE ✗
     
  A. "les hommes et les outils technologiques ne feront 
      plus qu'un" (humans and tech tools will become one)
     → Paragraph 2: "êtres humains et machines non 
       biologiques pourraient avoir si parfaitement fusionné 
       que les différences entre eux seraient devenues 
       négligeables"
     → FUSION + negligible differences = "ne feront plus 
       qu'un" (will become one) → STRONG MATCH ✓
     → KEEP ⚠️
     
  C. "Les outils technologiques seront supérieurs en 
      intelligence aux hommes" (Tech tools will be 
      superior in intelligence to humans)
     → The text says "l'intelligence humaine, améliorée, 
       pourrait commencer à s'étendre dans l'univers"
     → This is about ENHANCED HUMAN intelligence, not 
       machines being SUPERIOR to humans
     → BUT: the concept of "singularity" (paragraph 3) 
       is often associated with machines surpassing humans
     → SOME indirect support → KEEP ⚠️

  RESULT: A and C survive. Two-way tie.

Apply the Degree-of-Fit Scoring:

  → This captures the CENTRAL thesis ✓

  → "Hommes" = "êtres humains" ✓

  → It says "l'intelligence humaine, améliorée" = HUMAN

  → The direction is: humans get SMARTER (with tech help)

  → Kurzweil's vision (as presented in THIS text) is

  CRITERION 3: PRECISION                        [0/2]
  → "Supérieurs en intelligence" (superior in intelligence)
    → The text says "améliorée" (enhanced/improved)

  → The text says differences become "négligeables"

  → Vocabulary contradicts the text

✅ WHAT YOU SHOULD DO

DO 1: ALWAYS COMPLETE ELIMINATION BEFORE SCORING

THE CORRECT SEQUENCE:
                                                          
  STEP 1: Read all four options
  STEP 2: ELIMINATE options with clear disqualifiers
          (contradicts text, invented info, wrong tone)
  STEP 3: COUNT surviving options
  
  IF 1 SURVIVES → Select it (no scoring needed)
  IF 2 SURVIVE  → Apply scoring system to both
  IF 3 SURVIVE  → Apply scoring to all three
                   (rare — usually means you missed 
                    an elimination)
  IF 0 SURVIVE  → Re-read the text section; you 
                   eliminated too aggressively
                   (Chapter 13 re-examination method)

📌 Do NOT jump to scoring with all four options. Elimination first reduces the workload. Scoring is for the FINAL decision only.

DO 2: SCORE CRITERION 1 (TEXTUAL PROOF) FIRST — IT IS DECISIVE

PRIORITY ORDER OF CRITERIA:
                                                          
  1st: TEXTUAL PROOF (0-3 points)
  → This alone often decides the winner
  → If one option has 3/3 and the other has 0/3,
     you don't need to score the rest
  
  2nd: NO ADDED INFORMATION (0-2 points)
  → The second most decisive criterion
  → Invented content = automatic disqualifier
  
  3rd: PRECISION (0-2 points)
  → Catches subtle vocabulary shifts
  
  4th: COMPLETENESS (0-2 points)
  → Distinguishes partial from complete answers
  
  5th: QUESTION ALIGNMENT (0-1 point)
  → Usually both surviving options score 1/1 here
  → Rarely the deciding factor (Chapter 19 explores 
     the exception)

DO 3: WRITE THE SCORE DOWN (ON EXAM, IN MARGINS)

EXAM MARGIN NOTES:
                                                          
  Q26: B vs C?
  
  B: proof 3, complete 2, precise 2 = STRONG
  C: proof 1, complete 0, precise 0 = WEAK
  
  → B ✓
  
  (This takes 15 seconds and prevents second-guessing)

You do NOT need to score all five criteria with exact numbers on the exam. A quick assessment of the top three criteria is usually enough:

QUICK SCORING (exam speed version):
                                                          
  For each surviving option, ask THREE questions:
  
  ① "Can I POINT to specific lines that support this?"
     YES → strong    NO → weak
  
  ② "Does this answer ADD anything the text doesn't say?"
     NO → strong     YES → weak
  
  ③ "Does the answer use words that MATCH the text?"
     YES → strong    NO → weak
  
  If one option gets 3 "strong" and the other gets 
  1 "strong" and 2 "weak" → decision made in 20 seconds.

DO 4: LOOK FOR THE "KILLER WORD" — THE SINGLE TERM THAT BREAKS THE TIE

In every tie situation we analyzed, there was one word that decided everything:

THE KILLER WORDS IN OUR TIE SITUATIONS:
═══════════════════════════════════════════════════════════

  Q24: "réalisations" (achievements) 
       vs "évolutions" (developments)
       → Réalisations = done → WRONG (these are predictions)
       → Évolutions = proposed developments → RIGHT
       
  Q25: "supérieurs" (superior) 
       vs "ne feront plus qu'un" (will become one)
       → Supérieurs = domination → WRONG (text says fusion)
       → Ne feront plus qu'un = fusion → RIGHT
       
  Q26: "bonheur de tous" (happiness for all) 
       vs "dépollution" (depollution)
       → Bonheur = invented → WRONG (not in text)
       → Dépollution = "annuler la pollution" → RIGHT
       
  Q27: "idéalistes et réalistes" (idealists and realists)
       vs "scientifiques et métaphysiques"
       → Idéalistes/réalistes = added framework → WRONG
       → Scientifiques/métaphysiques = text's own 
         categories → RIGHT

═══════════════════════════════════════════════════════════
  
  PATTERN: The killer word is almost always in the 
  WRONG answer. It's the word that SEEMS to fit but 
  actually DISTORTS the text's meaning.
  
  STRATEGY: When two answers seem equal, find the word 
  in each that MOST differs from the text. The answer 
  with the MORE FAITHFUL vocabulary wins.

❌ WHAT YOU SHOULD NOT DO

MISTAKE 1: CHOOSING THE LONGER OR MORE DETAILED ANSWER

WRONG LOGIC:
  "Option C has more words and sounds more specific, 
   so it must be the better answer."
   
REALITY:
  Q26 Option A: "la vie éternelle et le bonheur de tous"
  → Sounds specific (eternal life! happiness for all!)
  → But "bonheur de tous" is INVENTED
  
  Q26 Option B: "l'intelligence artificielle et la dépollution"
  → Sounds simpler
  → But BOTH terms are IN the text
  
  SIMPLER + TEXTUALLY SUPPORTED > DETAILED + PARTIALLY INVENTED
  
  Length and detail are NOT indicators of correctness.
  Textual proof IS.

MISTAKE 2: CHOOSING BASED ON OUTSIDE KNOWLEDGE

WRONG LOGIC:
  "I know that AI singularity means machines surpassing 
   humans, so C must be right for Q25."
   
REALITY:
  Your knowledge about AI is IRRELEVANT.
  The question asks about what THIS TEXT says about 
  Kurzweil's PRINCIPALE théorie.
  
  THIS TEXT says: "parfaitement fusionné... différences 
  négligeables" = FUSION, not superiority.
  
  Your outside knowledge about singularity may be 
  CORRECT IN THE REAL WORLD but WRONG FOR THIS QUESTION.
  
  RULE: The text is your ONLY source of truth.
  Score against the text, not against your brain.

MISTAKE 3: CHANGING YOUR ANSWER AFTER SCORING

WRONG PATTERN:
  1. You score B at 10/10 and C at 3/10
  2. You select B
  3. You re-read the question
  4. You think "but C SOUNDS more like what they'd ask..."
  5. You change to C
  6. You get it WRONG
  
  THIS IS THE "SECOND-GUESS SPIRAL":
  
  Scoring decision (rational) → Gut feeling (irrational)
  → Change answer → WRONG
  
  FIX: Once you have scored and decided, MOVE ON.
  The scoring system exists to PREVENT gut-feeling errors.
  Trust the system. Do not override it with feelings.

MISTAKE 4: SPENDING TOO LONG ON ONE TIE

TIME MANAGEMENT:
                                                          
  Elimination:        30-45 seconds per question
  Scoring (if needed): 30-45 seconds additional
  TOTAL per question:  60-90 seconds maximum
  
  If you spend 3+ minutes on one tie, you steal time 
  from other questions that might be EASIER.
  
  EMERGENCY PROTOCOL:
  If after 90 seconds you STILL can't decide:
  1. Pick the option with MORE direct textual quotes
  2. Mark the question for review if time permits
  3. MOVE ON immediately
  
  A systematic guess based on textual proof > 
  a panicked guess based on nothing.

⚠️ COMMON PITFALLS AND FAILURE MODES

PITFALL 1: THE "BOTH ARE RIGHT" PARALYSIS

SYMPTOM: You convince yourself both options are equally 
         correct and freeze.

REALITY: In a well-designed exam, one answer is ALWAYS 
         better than the other. The difference might be 
         subtle, but it exists.
         
         Common sources of the difference:
         → One word is wrong ("réalisations" vs "évolutions")
         → One adds information ("bonheur de tous" = invented)
         → One answers a different question ("anticipation 
           de l'avenir" = describes Kurzweil, not his predictions)
         → One uses outside knowledge ("supérieurs" = AI concept 
           not in THIS text)
         
FIX: If you feel paralyzed, focus ONLY on Criterion 1 
     (Textual Proof). Ask: "Which option can I prove 
     with a SPECIFIC LINE from the text?"
     
     The one you can POINT TO in the text = the answer.
     The one you "feel" is right but can't prove = the trap.

PITFALL 2: THE SOPHISTICATION BIAS

SYMPTOM: You choose the answer that sounds most 
         "academic" or "intellectual."

EXAMPLES:
  "ravive l'éternel conflit entre idéalistes et réalistes"
  → Sounds like a philosophy class → WRONG
  
  "renvoie à des questions scientifiques et métaphysiques"
  → Sounds straightforward → RIGHT
  
  "la technologie humanisée et l'anticipation de l'avenir"
  → Sounds like a TED talk → WRONG
  
  "l'intelligence artificielle et la dépollution"
  → Sounds like a simple summary → RIGHT
  
  PATTERN: The wrong answer often sounds MORE impressive 
  than the right one.
  
  WHY: Exam writers know students are attracted to 
  sophisticated-sounding language. They use this as bait.
  
  FIX: Ignore how the answer SOUNDS.
  Score how it MATCHES THE TEXT.

PITFALL 3: THE PARTIAL MATCH TRAP

SYMPTOM: You see ONE word that matches the text and 
         assume the whole answer is correct.

EXAMPLE: Q26 Option A
  "la vie éternelle et le bonheur de tous"
  
  You see "vie éternelle" → "vivre éternellement" ✓
  Your brain says: "Match! This is correct!"
  You stop reading and select A.
  
  BUT: "bonheur de tous" has ZERO text support.
  Half the answer is correct, half is invented.
  HALF CORRECT = WRONG.
  
  FIX: ALWAYS check BOTH parts of a compound answer.
  An answer with "X et Y" requires BOTH X and Y 
  to be supported by the text.
  
  ┌──────────────────────────────────────────────┐
  │  COMPOUND ANSWER VERIFICATION:               │
  │                                              │
  │  "X et Y" → Is X in the text? ✓ or ✗        │
  │           → Is Y in the text? ✓ or ✗         │
  │                                              │
  │  Both ✓ = potentially correct                │
  │  One ✓ one ✗ = WRONG (partially fabricated)  │
  │  Both ✗ = definitely wrong                   │
  └──────────────────────────────────────────────┘

PITFALL 4: SCORING DRIFT — UNCONSCIOUSLY FAVORING YOUR FIRST INSTINCT

SYMPTOM: You had an initial "feeling" that B was correct.
         When you score, you unconsciously give B higher 
         scores to confirm your feeling.

THIS IS CONFIRMATION BIAS IN SCORING.

FIX: Score the option you DON'T instinctively prefer FIRST.
     
     If your gut says B, score C first.
     Give C the most generous interpretation possible.
     THEN score B with the same rigor.
     
     If C still scores lower after generous interpretation,
     your gut was right AND you have proof.
     
     If C scores HIGHER when you're fair to it,
     your gut was wrong and the system saved you.

🔗 BRIDGE TO CHAPTER 19

The scoring system solves most ties. But there is one tie type we identified but did NOT fully explore: Type 2 — "True But Wrong Question."

This is the most devious trap in advanced exam design. An answer can be:

  • ✅ Factually accurate about the text
  • ✅ Well-supported by textual evidence
  • ✅ Precisely worded
  • ❌ BUT it answers a DIFFERENT question than the one asked

In Q26, we saw a hint of this: Option C's "anticipation de l'avenir" describes what Kurzweil DOES, not what he PREDICTS. The question asked about predictions, not about his methods. This scope mismatch scored 0/1 on Criterion 5 (Question Alignment).

Chapter 19: "The Scope Trap: Answers That Are True But Don't Answer THIS Question" makes Criterion 5 the CENTRAL focus. It explores the most frustrating exam experience: when you are CERTAIN an answer is correct because it IS true — but it is true about the WRONG thing. Scope alignment is the most overlooked skill in comprehension, and Chapter 19 gives you the tools to master it.

📌 Chapter 18 taught you to SCORE between two plausible answers. Chapter 19 will teach you to DETECT the most dangerous distractor: the one that is TRUE but irrelevant.
