🧠 THE PROBLEM: THE MOST DANGEROUS DISTRACTOR IN EXISTENCE

Chapter 18 introduced the Degree-of-Fit Scoring System with five criteria. The first four criteria — Textual Proof, Completeness, Precision, No Added Information — all evaluate whether the answer matches the text. They ask: "Is this statement TRUE about the text?"

But Criterion 5 — Question Alignment — asks something fundamentally different:

"Does this TRUE statement answer the SPECIFIC question being asked?"

This is where the scope trap lives. And it is, without exaggeration, the single most effective weapon exam designers have against you.

Here is why it is so dangerous:

THE SCOPE TRAP MECHANISM:
═══════════════════════════════════════════════════════════

  NORMAL DISTRACTOR (Chapters 6-8):
  ┌─────────────────────────────────┐
  │ The answer is FALSE.            │
  │ It contradicts or invents.      │
  │ You check the text → not there. │
  │ You eliminate it → EASY.        │
  └─────────────────────────────────┘
  
  SCOPE TRAP DISTRACTOR (Chapter 19):
  ┌─────────────────────────────────┐
  │ The answer is TRUE.             │
  │ It IS in the text.              │
  │ You check the text → it's there.│
  │ You feel CONFIDENT.             │
  │ You select it.                  │
  │ You get it WRONG.               │
  │                                 │
  │ WHY? Because it answers a       │
  │ DIFFERENT question than the     │
  │ one that was asked.             │
  └─────────────────────────────────┘

  YOUR INTERNAL EXPERIENCE:
  
  "I KNOW this is in the text! I can POINT to the exact 
   line! How can this be wrong?!"
   
  → It is not wrong as a STATEMENT.
  → It is wrong as an ANSWER TO THIS QUESTION.
  
  STATEMENT TRUTH ≠ ANSWER CORRECTNESS

═══════════════════════════════════════════════════════════

An analogy to make this crystal clear:

ANALOGY: THE WRONG ADDRESS

  Question: "Where do you LIVE?"
  
  Answer A: "I live in Paris."           → ANSWERS the question ✓
  Answer B: "I was born in Lyon."        → TRUE about you ✓
                                           but answers WHERE 
                                           YOU WERE BORN,
                                           not where you LIVE ✗
  Answer C: "I work in Marseille."       → TRUE about you ✓
                                           but answers WHERE 
                                           YOU WORK,
                                           not where you LIVE ✗
  
  B and C are both TRUE FACTS about you.
  But neither ANSWERS the question "Where do you LIVE?"
  
  On an exam, B and C would be SCOPE TRAPS.
  They are correct information placed under the wrong question.

🔍 HOW TO APPROACH IT: THE SCOPE ANALYSIS FRAMEWORK

The solution is a two-step process that must happen BEFORE you even look at answer choices:

THE TWO-STEP SCOPE LOCK:
═══════════════════════════════════════════════════════════

  STEP 1: DISSECT THE QUESTION
  → Before reading ANY answer option, break the question 
    into its EXACT components
  → Identify WHAT is being asked
  → Identify the SCOPE (boundaries) of the question

  STEP 2: BUILD THE ANSWER TEMPLATE
  → Before reading ANY answer option, predict what a 
    correct answer MUST look like
  → Define what it MUST contain
  → Define what it MUST NOT contain
  → This template becomes your FILTER for evaluating options

═══════════════════════════════════════════════════════════

Let me demonstrate this with every question across all three texts.

SCOPE DISSECTION: TEXTE 101, QUESTION 24

QUESTION: "Cet article a pour but..."
(This article aims to...)

STEP 1: DISSECT THE QUESTION
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  SUBJECT: "Cet article"                                 │
│  → The ARTICLE as a whole                               │
│  → NOT Kurzweil's theory                                │
│  → NOT one paragraph                                    │
│  → The ENTIRE article's purpose                         │
│                                                         │
│  VERB: "a pour but"                                     │
│  → Aims to / has the PURPOSE of                         │
│  → Asking about AUTHOR'S INTENTION                      │
│  → NOT about content details                            │
│  → NOT about Kurzweil's intentions                      │
│                                                         │
│  SCOPE BOUNDARIES:                                      │
│  → Must describe what the JOURNALIST intended           │
│  → Must cover the WHOLE article, not one part           │
│  → Must be about PURPOSE (why written), not             │
│    CONTENT (what's inside)                              │
│                                                         │
└─────────────────────────────────────────────────────────┘

STEP 2: BUILD THE ANSWER TEMPLATE
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  A CORRECT ANSWER MUST:                                 │
│  ✓ Describe the journalist's PURPOSE                    │
│  ✓ Use an appropriate "purpose verb"                    │
│    (informer, présenter, dénoncer, convaincre, etc.)    │
│  ✓ Cover the article's OVERALL scope                    │
│                                                         │
│  A CORRECT ANSWER MUST NOT:                             │
│  ✗ Describe Kurzweil's predictions in detail            │
│  ✗ Focus on only one paragraph's content                │
│  ✗ Misrepresent the journalist's stance                 │
│                                                         │
│  TEMPLATE: "The article aims to [PURPOSE VERB]          │
│   about [BROAD TOPIC DESCRIPTION]"                      │
│                                                         │
└─────────────────────────────────────────────────────────┘

Now apply the template to the options:

OPTION A: "de dénoncer les thèses avant-gardistes 
           publiées par un chercheur"

  TEMPLATE CHECK:
  → Purpose verb: "dénoncer" (denounce) 
  → Does the journalist DENOUNCE? 
  → The text uses neutral language throughout
  → No condemnation vocabulary
  → FAILS template: wrong purpose verb ✗
  → ALSO: "publiées" (published) — the text doesn't 
    say Kurzweil published these, just that he says them
  → ELIMINATION TYPE: Wrong purpose + wrong detail


OPTION B: "de convaincre des avantages de la future 
           révolution technologique"

  TEMPLATE CHECK:
  → Purpose verb: "convaincre" (convince)
  → Does the journalist try to CONVINCE us?
  → The text uses conditional tense, distancing language
  → "Selon ses estimations" = reporting, not advocating
  → FAILS template: wrong purpose verb ✗
  → ALSO: "avantages" — the text doesn't emphasize advantages
  → ELIMINATION TYPE: Wrong purpose + added positive framing


OPTION C: "d'informer des évolutions technologiques 
           avancées par un scientifique"

  TEMPLATE CHECK:
  → Purpose verb: "informer" (inform)
  → Does the journalist INFORM? YES ✓
  → Topic: "évolutions technologiques" (tech developments)
  → Matches the article's content scope ✓
  → "Avancées par un scientifique" (proposed by a scientist)
  → Correctly attributes to Kurzweil ✓
  → PASSES template on ALL counts ✓


OPTION D: "de présenter des réalisations révolutionnaires 
           d'une vedette de la science"

  TEMPLATE CHECK:
  → Purpose verb: "présenter" (present)
  → Acceptable purpose verb ✓
  → BUT: "réalisations" (achievements)
  → The article presents PREDICTIONS, not achievements
  → SCOPE VIOLATION: "réalisations" changes the content 
    from FUTURE predictions to PAST accomplishments
  → FAILS template: wrong content description ✗
  → ELIMINATION TYPE: ★ SCOPE TRAP ★
    The word "présenter" fits, "vedette" fits,
    but "réalisations" shifts the SCOPE from predictions 
    to achievements — answering "what has he DONE?" 
    instead of "what does the article AIM TO DO?"

📌 Option D is the CLASSIC scope trap. Two out of three elements are correct ("présenter" + "vedette de la science"). But the third element ("réalisations") shifts the scope from what the article DOES (inform about predictions) to what Kurzweil supposedly DID (accomplish revolutionary things). The question asks about the ARTICLE'S purpose, and D misrepresents the article's content.

SCOPE DISSECTION: TEXTE 101, QUESTION 25

QUESTION: "La principale théorie de Ray Kurzweil affirme que..."
(The main theory of Ray Kurzweil states that...)

STEP 1: DISSECT THE QUESTION
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  SUBJECT: "La principale théorie"                       │
│  → The MAIN theory                                      │
│  → NOT a secondary detail                               │
│  → NOT all his predictions combined                     │
│  → The CENTRAL, overarching idea                        │
│                                                         │
│  POSSESSOR: "de Ray Kurzweil"                           │
│  → HIS theory                                           │
│  → NOT the journalist's interpretation                  │
│  → NOT other scientists' views                          │
│                                                         │
│  VERB: "affirme que"                                    │
│  → States that / claims that                            │
│  → Looking for what HE ASSERTS                          │
│                                                         │
│  SCOPE BOUNDARIES:                                      │
│  → Must be the CENTRAL idea (not a detail)              │
│  → Must be KURZWEIL'S claim (not a side effect)         │
│  → Must be what the THEORY STATES (not what we          │
│    can infer from it)                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘

STEP 2: BUILD THE ANSWER TEMPLATE
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  A CORRECT ANSWER MUST:                                 │
│  ✓ Describe the CENTRAL prediction                      │
│  ✓ Be something KURZWEIL claims                         │
│  ✓ Be the MAIN idea (unifying multiple predictions)     │
│                                                         │
│  A CORRECT ANSWER MUST NOT:                             │
│  ✗ Describe a SECONDARY prediction                      │
│  ✗ Describe a CONSEQUENCE of the theory                 │
│  ✗ Describe something the TEXT IMPLIES but Kurzweil     │
│    doesn't specifically claim                           │
│  ✗ Describe a DIFFERENT person's interpretation         │
│                                                         │
│  TEMPLATE: "Kurzweil's main theory is that              │
│   [CENTRAL CLAIM about humans and technology]"          │
│                                                         │
└─────────────────────────────────────────────────────────┘

Apply the template:

SCOPE TRAP ANALYSIS FOR Q25:
═══════════════════════════════════════════════════════════

  OPTION A: "les hommes et les outils technologiques 
             ne feront plus qu'un"
  
  SCOPE CHECK:
  → Is this the MAIN theory? 
  → Text: "êtres humains et machines... parfaitement 
    fusionné... différences négligeables"
  → Fusion of humans and machines IS the central, 
    unifying prediction
  → Brain upload → fusion
  → Enhanced intelligence → fusion
  → Expanding in universe → result of fusion
  → EVERYTHING leads to or follows from FUSION
  → This IS the main theory ✓
  → PASSES scope template ✓

─────────────────────────────────────────────────────────

  OPTION B: "La technologie dépossédera l'homme de 
             certaines de ses qualités"
  
  SCOPE CHECK:
  → Is this what Kurzweil CLAIMS?
  → The text says technology will ENHANCE ("améliorée"), 
    not DIMINISH ("dépossédera")
  → This CONTRADICTS Kurzweil's theory
  → Not a scope trap — simply FALSE ✗
  → FAILS on content (not scope)

─────────────────────────────────────────────────────────

  OPTION C: "Les outils technologiques seront supérieurs 
             en intelligence aux hommes"
  
  SCOPE CHECK:
  → Is this what Kurzweil claims IN THIS TEXT?
  → The text describes FUSION, not SUPERIORITY
  → BUT: This is related to the concept of singularity
  → It might be TRUE about Kurzweil's ideas IN GENERAL
  → However, THIS TEXT presents his theory as FUSION
  
  ★ THIS IS A SCOPE TRAP ★
  
  → It answers: "What might singularity mean in general?"
  → It does NOT answer: "What does THIS TEXT say is 
    Kurzweil's MAIN theory?"
  → The text's main theory = fusion (becoming one)
  → NOT machines being superior
  
  SCOPE VIOLATION TYPE: OUTSIDE KNOWLEDGE SCOPE
  → True about the general topic
  → NOT true about THIS TEXT's specific presentation
  → FAILS scope template ✗

─────────────────────────────────────────────────────────

  OPTION D: "L'humanité utilisera la technologie pour 
             améliorer son patrimoine génétique"
  
  SCOPE CHECK:
  → Does THIS TEXT say Kurzweil predicts genetic improvement?
  → "Génie génétique" appears in paragraph 3 as a 
    CONTROVERSIAL THEME, not as a specific prediction
  → "Patrimoine génétique" appears in TEXTE 102 (salmon!), 
    not in Texte 101
  
  ★ THIS IS A CROSS-TEXT SCOPE TRAP ★
  
  → It borrows vocabulary from a DIFFERENT text
  → "Patrimoine génétique" is from the salmon article
  → Placed here, it SOUNDS relevant because "génétique" 
    appears in both texts
  → But in Texte 101, "génie génétique" is listed as 
    a debate topic, NOT as Kurzweil's prediction
  
  SCOPE VIOLATION TYPE: CROSS-TEXT CONTAMINATION
  → True about Texte 102 (salmon genetic heritage)
  → NOT true about Texte 101 (Kurzweil's theory)
  → FAILS scope template ✗

═══════════════════════════════════════════════════════════

📌 CRITICAL DISCOVERY: Option D is a CROSS-TEXT scope trap. It takes vocabulary from Texte 102 ("patrimoine génétique") and plants it in a question about Texte 101. If you are doing a multi-text exam and your brain is holding vocabulary from all texts simultaneously, this trap catches you.

SCOPE DISSECTION: TEXTE 101, QUESTION 26

QUESTION: "Les changements majeurs prédits par Ray Kurzweil sont..."
(The major changes predicted by Ray Kurzweil are...)

STEP 1: DISSECT THE QUESTION
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  KEY WORD 1: "changements" (changes)                    │
│  → Must be CHANGES in the world                         │
│  → NOT qualities of Kurzweil                            │
│  → NOT descriptions of his method                       │
│  → Things that will be DIFFERENT in the future          │
│                                                         │
│  KEY WORD 2: "majeurs" (major)                          │
│  → Must be BIG, significant changes                     │
│  → NOT minor details                                    │
│  → The most important ones                              │
│                                                         │
│  KEY WORD 3: "prédits" (predicted)                      │
│  → Must be PREDICTIONS (future, uncertain)              │
│  → NOT current realities                                │
│  → NOT accomplished facts                               │
│                                                         │
│  KEY WORD 4: "par Ray Kurzweil"                         │
│  → Must be HIS predictions specifically                 │
│  → NOT general scientific consensus                     │
│                                                         │
│  SCOPE BOUNDARIES:                                      │
│  → Changes (not methods, not qualities, not activities) │
│  → Major (not minor details)                            │
│  → Predicted (not accomplished, not current)            │
│  → By Kurzweil (not by others, not by the journalist)   │
│                                                         │
└─────────────────────────────────────────────────────────┘

Now watch how each wrong answer violates a DIFFERENT scope boundary:

SCOPE VIOLATION MAP FOR Q26:
═══════════════════════════════════════════════════════════

  OPTION A: "la vie éternelle et le bonheur de tous"

  WORD-BY-WORD SCOPE CHECK:
  
  "la vie éternelle" 
  → Is this a CHANGE? Yes — living forever changes humanity ✓
  → Is it MAJOR? Yes ✓
  → Is it PREDICTED by Kurzweil? 
    "vivre éternellement sans vieillir" → Yes ✓
  → FIRST HALF PASSES SCOPE ✓
  
  "le bonheur de tous"
  → Is this a CHANGE? It would be, if predicted ✓
  → Is it PREDICTED by Kurzweil? 
    → NOWHERE in the text does Kurzweil predict happiness ✗
  → SECOND HALF FAILS SCOPE: INVENTED CONTENT
  
  SCOPE VIOLATION: Half-scope match + fabrication
  → "Vie éternelle" is IN scope
  → "Bonheur de tous" is OUT of scope (not in text at all)
  
  TRAP TYPE: PARTIAL SCOPE MATCH
  → One half is correctly scoped
  → One half is completely invented
  → Your brain latches onto the correct half and 
    ignores the invented half

─────────────────────────────────────────────────────────

  OPTION B: "l'intelligence artificielle et la dépollution"

  WORD-BY-WORD SCOPE CHECK:
  
  "l'intelligence artificielle"
  → Is this a CHANGE? Yes — AI would change the world ✓
  → Is it MAJOR? Yes ✓
  → Is it PREDICTED by Kurzweil?
    → Paragraph 3 mentions it in the debate context ✓
    → Paragraph 2 describes enhanced intelligence ✓
  → FIRST HALF PASSES SCOPE ✓
  
  "la dépollution"
  → Is this a CHANGE? Yes — removing pollution changes 
    the world ✓
  → Is it MAJOR? Yes ✓
  → Is it PREDICTED by Kurzweil?
    → "minuscules robots annuleront les effets de la 
      pollution" → YES ✓
  → SECOND HALF PASSES SCOPE ✓
  
  BOTH HALVES PASS ALL SCOPE BOUNDARIES ✓✓
  → Changes ✓ + Major ✓ + Predicted ✓ + By Kurzweil ✓
  
  ANSWER: B ✓

─────────────────────────────────────────────────────────

  OPTION C: "la technologie humanisée et l'anticipation 
             de l'avenir"

  WORD-BY-WORD SCOPE CHECK:
  
  "la technologie humanisée"
  → Is this a CHANGE? Debatable — it's more of a 
    DESCRIPTION of technology ⚠️
  → Is it PREDICTED by Kurzweil? The text says 
    "fusionné" not "humanisée" — interpretation, 
    not direct prediction ⚠️
  → FIRST HALF: BORDERLINE SCOPE
  
  "l'anticipation de l'avenir"
  → Is this a CHANGE? ✗ NO!
  → "Anticipation de l'avenir" = anticipating the future
  → This is what KURZWEIL DOES (his activity)
  → It is NOT a change he PREDICTS in the world
  → The question asks: what changes will happen?
  → "Anticipating the future" is not something that 
    HAPPENS TO THE WORLD — it's something Kurzweil does
  
  ★ THIS IS THE PUREST SCOPE TRAP IN ALL OUR TEXTS ★
  
  SCOPE VIOLATION: ACTIVITY vs CHANGE
  → The question asks about PREDICTED CHANGES 
    (things that will be different)
  → "Anticipation de l'avenir" describes KURZWEIL'S 
    ACTIVITY (what he does as a futurist)
  → It is TRUE that Kurzweil anticipates the future
  → But that is NOT a predicted CHANGE in the world
  
  ANALOGY:
  Question: "What weather changes does the forecaster predict?"
  Wrong answer: "Forecasting and watching the sky"
  → Forecasting is what the FORECASTER does
  → It's not a WEATHER CHANGE he predicts
  
  FAILS SCOPE: answers "what does Kurzweil DO?" 
  instead of "what CHANGES does he predict?" ✗

─────────────────────────────────────────────────────────

  OPTION D: "La robotisation du quotidien et la dictature 
             des machines"

  WORD-BY-WORD SCOPE CHECK:
  
  "la robotisation du quotidien"
  → Is this PREDICTED by Kurzweil in this text?
  → The text mentions "minuscules robots" but for 
    POLLUTION, not for daily life
  → "du quotidien" (of daily life) is ADDED ✗
  → SCOPE VIOLATION: adds "daily life" not in text
  
  "la dictature des machines"
  → Is this PREDICTED by Kurzweil?
  → The text says FUSION with "negligible differences"
  → Fusion ≠ dictatorship
  → "Dictature" is the OPPOSITE of fusion
  → SCOPE VIOLATION: inverts the text's meaning ✗
  
  DOUBLE SCOPE FAILURE ✗✗

SCOPE DISSECTION: TEXTE 101, QUESTION 27

QUESTION: "La polémique suscitée par l'ensemble de 
           ces prédictions..."
(The controversy sparked by all of these predictions...)

STEP 1: DISSECT THE QUESTION
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  SUBJECT: "La polémique" (the controversy/debate)       │
│  → Must be about the DEBATE ITSELF                      │
│  → NOT about the predictions themselves                 │
│  → NOT about Kurzweil himself                           │
│  → About what the CONTROVERSY involves/relates to       │
│                                                         │
│  QUALIFIER: "suscitée par l'ensemble de ces prédictions"│
│  → Sparked BY these predictions                         │
│  → The predictions CAUSE the controversy                │
│  → We need to describe the CONTROVERSY, not the         │
│    predictions                                          │
│                                                         │
│  SCOPE BOUNDARIES:                                      │
│  → About the CONTROVERSY (not the predictions)          │
│  → About what it RELATES TO or INVOLVES                 │
│  → About the NATURE of the debate                       │
│                                                         │
└─────────────────────────────────────────────────────────┘

STEP 2: BUILD THE ANSWER TEMPLATE
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  A CORRECT ANSWER MUST:                                 │
│  ✓ Describe the controversy's NATURE or SCOPE           │
│  ✓ Explain what DOMAINS the debate touches              │
│  ✓ Match paragraph 3's description of the debate        │
│                                                         │
│  A CORRECT ANSWER MUST NOT:                             │
│  ✗ Describe what the predictions ARE                    │
│  ✗ Predict what will HAPPEN TO the controversy          │
│    (unless the text says so)                            │
│  ✗ Add philosophical frameworks not in the text         │
│                                                         │
└─────────────────────────────────────────────────────────┘
SCOPE TRAP ANALYSIS FOR Q27:
═══════════════════════════════════════════════════════════

  OPTION A: "mettra un terme aux controverses traditionnelles"
  (will put an end to traditional controversies)
  
  SCOPE CHECK:
  → Does the text say the controversy will END?
  → NO — "croissance exponentielle" = GROWTH, not ending
  → DIRECTLY contradicts the text
  → NOT a scope trap — simply FALSE ✗

─────────────────────────────────────────────────────────

  OPTION B: "ravive l'éternel conflit entre idéalistes 
             et réalistes"
  (revives the eternal conflict between idealists and realists)
  
  SCOPE CHECK:
  → "L'éternel conflit entre idéalistes et réalistes"
  → Is this what the TEXT describes?
  → The text describes divided REACTIONS (paragraph 1):
    "désaccord véhément / admiration éperdue / scepticisme"
  → And divided THEMES (paragraph 3):
    "thèmes controversés + concepts de science-fiction"
  → NEITHER of these is framed as "idealists vs realists"
  
  ★ SCOPE TRAP TYPE: PHILOSOPHICAL FRAMEWORK ADDITION ★
  
  → The answer IMPOSES an external interpretive framework
    ("idealists vs realists") onto the text
  → The text never uses these categories
  → The text's OWN categories are:
    → Reactions: agreement / skepticism / admiration
    → Content: science / metaphysics / science-fiction
  → "Idealists vs realists" is a DIFFERENT framework 
    that could describe many controversies
  → It does not specifically match THIS text's description
  
  SCOPE VIOLATION: FRAMEWORK SUBSTITUTION
  → The answer substitutes the text's specific framework 
    with a generic philosophical one ✗

─────────────────────────────────────────────────────────

  OPTION C: "renvoie à des questions scientifiques 
             et métaphysiques"
  (refers to scientific and metaphysical questions)
  
  SCOPE CHECK:
  → Does the text describe the controversy as involving 
    scientific and metaphysical questions?
  → Paragraph 3: "thèmes extrêmement controversés 
    (Dieu, la crise énergétique, le génie génétique)"
    → "Dieu" = God = METAPHYSICAL ✓
    → "crise énergétique" = SCIENTIFIC/POLICY ✓
    → "génie génétique" = SCIENTIFIC ✓
  → "concepts de science-fiction (l'intelligence artificielle, 
    la réalité virtuelle, l'ingénierie moléculaire)"
    → All SCIENTIFIC concepts ✓
  → "Scientifiques et métaphysiques" ACCURATELY CATEGORIZES 
    the text's own list ✓
  
  PASSES ALL SCOPE BOUNDARIES:
  → About the CONTROVERSY ✓ (what it "renvoie à")
  → About its NATURE ✓ (scientific + metaphysical)
  → MATCHES paragraph 3 ✓
  → Uses the text's OWN categories, not an imported framework ✓

─────────────────────────────────────────────────────────

  OPTION D: "s'éteindra avec le développement inéluctable 
             de la technologie"
  (will die out with the inevitable development of technology)
  
  SCOPE CHECK:
  → "S'éteindra" (will die out) contradicts 
    "croissance exponentielle" (exponential growth)
  → "Inéluctable" (inevitable) contradicts the 
    conditional "si le rythme continue" (IF the pace continues)
  → NOT a scope trap — DOUBLY contradicts the text ✗

SCOPE DISSECTION: TEXTE 102, QUESTION 1

QUESTION: "L'article explique..."
(The article explains...)

STEP 1: DISSECT THE QUESTION
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  SUBJECT: "L'article" (the article as a whole)          │
│  → Must cover the OVERALL content                       │
│  → NOT just one paragraph                               │
│                                                         │
│  VERB: "explique" (explains)                            │
│  → What does the article EXPLAIN?                       │
│  → Not what it mentions in passing                      │

└─────────────────────────────────────────────────────────┘

  The ENTIRE article describes HOW salmon is farmed

🔧 THE SCOPE TRAP TAXONOMY: ALL TYPES CLASSIFIED

Based on every example we have analyzed, here is the complete taxonomy of scope traps:

THE 7 TYPES OF SCOPE TRAPS:
═══════════════════════════════════════════════════════════

  TYPE 1: DETAIL ELEVATED TO MAIN TOPIC
  ─────────────────────────────────────
  → A minor detail from the text is presented as the 
    article's main subject
  → Example: "Les dangers du mercure" for an article 
    about salmon farming (mercury is one detail)
  → Example: "Au risque des frontières" for alpinism 
    (Italian side is one mention)
  → HOW TO DETECT: Ask "Is this what MOST of the text 
    is about, or just one sentence?"

─────────────────────────────────────────────────────────

  TYPE 2: CONTEXT CONFUSED WITH TOPIC
  ─────────────────────────────────────
  → Background information is presented as the main topic
  → Example: "Norway's trade superiority" when the article 
    is about farming PRACTICES (trade is context)
  → Example: Q24 option B "convaincre des avantages" 
    (the article provides context about tech, not advocacy)
  → HOW TO DETECT: Ask "Is this the SETUP or the 
    MAIN CONTENT?"

─────────────────────────────────────────────────────────

  TYPE 3: ACTIVITY CONFUSED WITH CHANGE
  ─────────────────────────────────────
  → What someone DOES is presented as what they PREDICT
  → Example: Q26 option C "l'anticipation de l'avenir"
    (anticipating IS what Kurzweil does, but it's not 
    a CHANGE he predicts in the world)
  → HOW TO DETECT: Ask "Is this something that HAPPENS 
    TO THE WORLD or something the PERSON DOES?"

─────────────────────────────────────────────────────────

  TYPE 4: OUTSIDE KNOWLEDGE PRESENTED AS TEXT CONTENT
  ─────────────────────────────────────────────────────
  → True information about the topic FROM OUTSIDE 
    the text is presented as an answer
  → Example: Q25 option C "machines supérieures en 
    intelligence" (true about AI generally, but not 
    what THIS TEXT says)
  → HOW TO DETECT: Ask "Can I find this SPECIFICALLY 
    in THIS text, or do I know it from elsewhere?"

─────────────────────────────────────────────────────────

  TYPE 5: CROSS-TEXT CONTAMINATION
  ─────────────────────────────────────
  → Vocabulary or concepts from a DIFFERENT text on the 
    same exam are planted in this question
  → Example: Q25 option D "patrimoine génétique" 
    (from Texte 102 about salmon, not Texte 101)
  → HOW TO DETECT: Ask "Is this vocabulary from THIS 
    text or from ANOTHER text I just read?"

─────────────────────────────────────────────────────────

  TYPE 6: FRAMEWORK SUBSTITUTION
  ─────────────────────────────────────
  → An external analytical framework is imposed on 
    the text's own categorization
  → Example: Q27 option B "idéalistes et réalistes" 
    (the text uses science/metaphysics categories, 
    not idealist/realist categories)
  → HOW TO DETECT: Ask "Does the TEXT use these 
    categories, or am I importing them?"

─────────────────────────────────────────────────────────

  TYPE 7: CONSEQUENCE CONFUSED WITH CAUSE
  ─────────────────────────────────────
  → A RESULT of the text's topic is presented as 
    the topic itself
  → Example: If Q1 for Texte 102 had an option like 
    "Les risques sanitaires liés à la consommation 
    de poisson" (health risks from eating fish)
    → Health risks are a CONSEQUENCE of farming practices
    → The article EXPLAINS practices, not health risks
  → HOW TO DETECT: Ask "Is this what the text 
    EXPLAINS or what RESULTS from what it explains?"

✅ WHAT YOU SHOULD DO

DO 1: DISSECT THE QUESTION BEFORE LOOKING AT ANSWERS

This is the single most important habit this chapter teaches:

THE QUESTION DISSECTION PROTOCOL:
═══════════════════════════════════════════════════════════

  BEFORE looking at A, B, C, D:
  
  1. READ the question stem carefully
  
  2. IDENTIFY the subject:
     → What is being asked ABOUT?
     → The article? A person? A theory? A debate?
  
  3. IDENTIFY the verb:
     → What are we asked to DO with it?
     → Explain? Describe? Identify? Evaluate?
  
  4. IDENTIFY the scope boundaries:
     → What MUST the answer be about?
     → What MUST it NOT be about?
  
  5. BUILD the answer template:
     → "A correct answer would look like..."
     → "A correct answer would NOT look like..."
  
  6. ONLY NOW read the options and match against template

═══════════════════════════════════════════════════════════

Practice this with EVERY question:

QUICK DISSECTION EXAMPLES:
                                                          
  "Cet article a pour but..."
  → Subject: THE ARTICLE
  → Verb: has the PURPOSE of
  → Scope: OVERALL purpose, not details
  → Template: "[purpose verb] about [broad topic]"

  "La principale théorie affirme que..."
  → Subject: THE MAIN THEORY
  → Verb: states/claims
  → Scope: CENTRAL idea, not secondary predictions
  → Template: "The main claim is that [central prediction]"

  "Les changements majeurs prédits sont..."
  → Subject: MAJOR CHANGES
  → Verb: predicted (are)
  → Scope: CHANGES in the world, MAJOR ones, PREDICTED
  → Template: "[big thing 1] and [big thing 2] that will happen"

  "La polémique suscitée..."
  → Subject: THE CONTROVERSY
  → Verb: (implied: involves/relates to)
  → Scope: About the DEBATE, not the predictions themselves
  → Template: "relates to [type of] questions/domains"

DO 2: USE THE "SWAP TEST" FOR SCOPE VERIFICATION

When you suspect a scope trap, apply the Swap Test:

THE SWAP TEST:
═══════════════════════════════════════════════════════════

  Take the answer option and place it as the answer 
  to a DIFFERENT question. If it fits better under 
  a different question, it's a scope trap.

  EXAMPLE 1:
  Q26 Option C: "la technologie humanisée et 
                 l'anticipation de l'avenir"
  
  → Does this answer Q26 ("changements majeurs prédits")?
    → "Anticipation de l'avenir" is not a predicted CHANGE
    → Weak fit ⚠️
  
  → Would this better answer: "Qu'est-ce qui caractérise 
    Ray Kurzweil?" (What characterizes Kurzweil?)
    → "Anticipation de l'avenir" = YES, this is what HE does
    → PERFECT fit for a different question ✓
  
  → SWAP TEST RESULT: This answer belongs to a DIFFERENT 
    question → SCOPE TRAP CONFIRMED ✗

─────────────────────────────────────────────────────────

  EXAMPLE 2:
  Q25 Option D: "L'humanité utilisera la technologie 
                 pour améliorer son patrimoine génétique"
  
  → Does this answer Q25 about Kurzweil's theory in Texte 101?
    → "Patrimoine génétique" is from Texte 102
    → Weak fit ⚠️
  
  → Would this better answer a question about Texte 102?
    → "Patrimoine génétique des saumons sauvages" is a 
      key concept in the salmon article
    → Better fit for a different text ✓
  
  → SWAP TEST RESULT: This answer belongs to a DIFFERENT 
    TEXT → CROSS-TEXT SCOPE TRAP CONFIRMED ✗

─────────────────────────────────────────────────────────

  EXAMPLE 3:
  Q20 Option D: "Au risque des frontières"
  
  → Does this answer Q20 (best title for alpinism article)?
    → Borders are mentioned once → Weak fit ⚠️
  
  → Would this better suit an article about border 
    conflicts in mountain regions?
    → PERFECT fit for that type of article ✓
  
  → SWAP TEST RESULT: This title belongs to a DIFFERENT 
    article → SCOPE TRAP CONFIRMED ✗

═══════════════════════════════════════════════════════════

📌 THE SWAP TEST RULE: If an answer fits a DIFFERENT question better than the one being asked, it is a scope trap. The correct answer fits THIS question BEST and would NOT work as well under a different question.

DO 3: CHECK EVERY WORD IN THE QUESTION STEM

Scope traps exploit the fact that students read questions QUICKLY and miss scope-defining words:

SCOPE-DEFINING WORDS AND THEIR IMPACT:
═══════════════════════════════════════════════════════════

  "PRINCIPALE théorie" (Q25)
  ─────────────────────
  → "Principale" = MAIN
  → You must find the CENTRAL theory, not a side prediction
  → If an answer describes a TRUE but SECONDARY prediction, 
    it fails the "principale" scope requirement

  "Changements MAJEURS" (Q26)
  ─────────────────────
  → "Majeurs" = MAJOR
  → Minor details don't qualify
  → Must be the BIGGEST predicted changes

  "L'ENSEMBLE de ces prédictions" (Q27)
  ─────────────────────
  → "L'ensemble" = ALL of / the totality of
  → The controversy is about ALL predictions together
  → Not about one specific prediction
  → The answer must capture the BROAD debate, not one aspect

  "Cet ARTICLE a pour but" (Q24)
  ─────────────────────
  → "Cet article" = THIS article
  → Not Kurzweil's work, not technology in general
  → Specifically what THIS journalist's article aims to do

  "Quel titre convient LE MIEUX" (Q20)
  ─────────────────────
  → "Le mieux" = THE BEST
  → Not "a possible title" but "the BEST title"
  → Multiple titles might work; you need the MOST fitting

═══════════════════════════════════════════════════════════

DO 4: APPLY THE "ZOOM LEVEL" CHECK

Every question operates at a specific zoom level. The answer must match that zoom level:

THE ZOOM LEVEL FRAMEWORK:
═══════════════════════════════════════════════════════════

  ZOOM LEVEL 1: MACRO (whole text)
  → "What is the article about?"
  → "What is the article's purpose?"
  → "What title fits best?"
  → Answers must cover the ENTIRE text
  → TRAP: Answers that zoom into ONE paragraph

  ZOOM LEVEL 2: THEME (central idea)
  → "What is the main theory?"
  → "What are the major changes?"
  → Answers must capture the CORE message
  → TRAP: Answers that capture a side detail

  ZOOM LEVEL 3: SPECIFIC (particular detail)
  → "What are the two problems mentioned?"
  → "What did this person specifically do?"
  → Answers must match EXACT text details
  → TRAP: Answers that are too general or vague

  ZOOM LEVEL 4: META (about the text's effect)
  → "What controversy does this create?"
  → "What reaction does this provoke?"
  → Answers must be about the TEXT'S IMPACT
  → TRAP: Answers that describe the text's CONTENT 
    instead of its IMPACT

─────────────────────────────────────────────────────────

  QUESTION → ZOOM LEVEL MAPPING:
  
  Q24 "article a pour but"        → LEVEL 1 (MACRO)
  Q25 "principale théorie"        → LEVEL 2 (THEME)
  Q26 "changements majeurs"       → LEVEL 2-3 (THEME-SPECIFIC)
  Q27 "polémique suscitée"        → LEVEL 4 (META)
  Q20 "quel titre"                → LEVEL 1 (MACRO)
  Q1  "l'article explique"        → LEVEL 1 (MACRO)
  
  SCOPE TRAP DETECTION:
  If the question is LEVEL 1 (macro) and the answer 
  is LEVEL 3 (specific detail) → SCOPE MISMATCH
  
  If the question is LEVEL 4 (meta/impact) and the answer 
  is LEVEL 2 (theme/content) → SCOPE MISMATCH

═══════════════════════════════════════════════════════════

Visual demonstration:

Q27 asks at ZOOM LEVEL 4 (META — about the controversy):

  ┌──────────────────────────────────────────────────┐
  │                                                  │
  │  LEVEL 4: What does the CONTROVERSY involve?     │
  │           → Scientific + metaphysical questions   │ ← CORRECT (C)
  │                                                  │
  │  LEVEL 2: What does KURZWEIL predict?            │
  │           → Fusion of humans and machines        │ ← WRONG ZOOM
  │                                                  │
  │  LEVEL 1: What is the ARTICLE about?             │
  │           → A futurist's predictions             │ ← WRONG ZOOM
  │                                                  │
  └──────────────────────────────────────────────────┘
  
  Option B "idéalistes et réalistes" SOUNDS like Level 4
  but actually imports an external framework → STILL wrong
  
  Option C "scientifiques et métaphysiques" IS Level 4
  and matches the text's own categories → CORRECT

❌ WHAT YOU SHOULD NOT DO

MISTAKE 1: SELECTING AN ANSWER BECAUSE "IT IS IN THE TEXT"

THE FUNDAMENTAL ERROR:
                                                          
  Your thinking: "I can find this in the text, 
                  therefore it must be the answer."
  
  REALITY: Being "in the text" is NECESSARY but 
           NOT SUFFICIENT.
  
  An answer must be:
  ① In the text                    (necessary)
  ② Correctly scoped to the question (necessary)
  ③ The BEST match among all options (necessary)
  
  ALL THREE must be true simultaneously.
  
  EXAMPLE:
  Q26 asks about "changements majeurs prédits"
  
  "L'anticipation de l'avenir" — Is this in the text?
  → YES, Kurzweil DOES anticipate the future ✓
  → But is it a PREDICTED CHANGE? NO ✗
  → It is in the text but WRONGLY SCOPED
  
  "I can find it in the text" = necessary condition MET
  "It answers THIS question" = necessary condition FAILED
  
  RESULT: Wrong answer selected despite finding text proof

MISTAKE 2: NOT READING THE QUESTION CAREFULLY ENOUGH

THE SPEED-READING TRAP:
                                                          
  What you READ:  "Les changements... Kurzweil..."
  What you THINK: "What does Kurzweil say?"
  
  What the question ACTUALLY asks:
  "Les changements MAJEURS PRÉDITS par Ray Kurzweil sont"
  
  You missed:
  → "Changements" = CHANGES (not activities, not methods)
  → "Majeurs" = MAJOR (not minor)
  → "Prédits" = PREDICTED (not accomplished)
  
  Each word you missed opens the door to a scope trap.
  
  FIX: Read the question stem TWICE before looking 
  at any option. On the second read, UNDERLINE the 
  scope-defining words.

MISTAKE 3: ASSUMING THE QUESTION ASKS WHAT YOU EXPECT

THE EXPECTATION TRAP:
                                                          
  You read the text about Kurzweil.
  You understand his predictions.
  You expect questions to ask: "What does he predict?"
  
  Q27 arrives: "La polémique suscitée par..."
  
  Your brain STILL thinks: "What does he predict?"
  You look for an answer about predictions.
  You find one that mentions idealists vs realists.
  "That sounds like a reaction to predictions!"
  You select it.
  
  BUT the question asks about the CONTROVERSY'S NATURE,
  not about the predictions.
  
  The correct answer describes WHAT DOMAINS the 
  controversy TOUCHES (science + metaphysics),
  not HOW PEOPLE REACT to predictions.
  
  FIX: Reset your brain for EACH question.
  Don't carry expectations from previous questions.
  Each question has its OWN scope.

MISTAKE 4: CONFUSING "RELATED TO THE TEXT" WITH "ANSWERS THE QUESTION"

HIERARCHY OF RELEVANCE:
                                                          
  LEVEL 1: "Related to the text's general topic"
  → Mercury is related to salmon farming
  → But "mercury dangers" ≠ article's main explanation
  
  LEVEL 2: "True about the text's content"
  → Kurzweil anticipates the future — TRUE
  → But "anticipation" ≠ a predicted CHANGE
  
  LEVEL 3: "Directly answers the specific question"
  → "Intelligence artificielle et dépollution" 
    ARE major predicted changes ✓
  
  STUDENTS SELECT AT LEVEL 1 OR 2.
  CORRECT ANSWERS LIVE AT LEVEL 3.
  
  The gap between "related to the text" and 
  "answers this specific question" is where 
  ALL scope traps live.

⚠️ COMMON PITFALLS AND FAILURE MODES

PITFALL 1: THE CROSS-TEXT VOCABULARY BLEED

SCENARIO: You have just read Texte 102 about salmon.
Your brain is full of: "patrimoine génétique," 
"pollution," "écosystème," "métaux lourds."

Now you read Texte 101 about Kurzweil.
Paragraph 3 mentions "génie génétique."

Q25 option D says: "améliorer son patrimoine génétique"

Your brain lights up: "Patrimoine génétique! 
I JUST saw this! It must be important!"

BUT: "patrimoine génétique" is from the SALMON text.
In the Kurzweil text, only "génie génétique" appears,
and only as a DEBATE TOPIC, not as a prediction.

Your brain CROSS-CONTAMINATED the two texts.

FIX: When switching between texts on an exam,
     mentally RESET. Ask yourself:
     "Am I thinking about THIS text or the previous one?"
     
     If a term feels familiar, check: 
     "Where did I see this? In THIS text or in ANOTHER?"

PITFALL 2: THE "SOUNDS RIGHT" CONFIRMATION LOOP

SCENARIO: You have a vague understanding of the text.
You read option B for Q27: "ravive l'éternel conflit 
entre idéalistes et réalistes."

Your reaction: "That sounds right. There IS a conflict 
in the text. Some people admire Kurzweil, others are 
skeptical. That's like idealists vs realists!"

You selected based on FEELING, not SCOPE CHECKING.

THE PROBLEM:
→ The text's division is about REACTIONS to predictions
  (admiration, skepticism, disagreement)
→ NOT about philosophical positions (idealism vs realism)
→ AND the question asks about the CONTROVERSY'S NATURE
  (what domains it touches), not about people's reactions
→ You answered: "How do people react?" (not asked)
→ Instead of: "What domains does the controversy touch?" (asked)

FIX: Never select based on "sounds right."
     Always verify: "Does this answer THIS SPECIFIC question?"
     The Swap Test from earlier is your safeguard.

PITFALL 3: THE ZOOM LEVEL MISMATCH BLINDSPOT

SCENARIO: Q20 asks for the BEST TITLE (Zoom Level 1: Macro)

You loved the dramatic story of Balmat's bivouac.
Option A: "La grande crevasse" SOUNDS dramatic and 
mountaineering-related.

You select it because it FEELS like it matches the 
exciting parts of the text.

THE PROBLEM:
→ "La grande crevasse" is Zoom Level 3 (specific detail)
→ The question needs Zoom Level 1 (whole text coverage)
→ Crevasses aren't even MENTIONED in the text
→ You zoomed INTO one dramatic element and forgot 
  the question asks about the WHOLE article

FIX: Before selecting a title, ask:
     "Does this cover Saussure AND Balmat AND Paradis 
      AND d'Angeville AND the stagnation period?"
     If it only covers ONE part → wrong zoom level.

PITFALL 4: THE "IT ANSWERS A BETTER QUESTION" SEDUCTION

THE MOST SUBTLE PITFALL:

Sometimes a scope trap answer is SO well-crafted that 
you think: "The question SHOULD have asked this instead."

EXAMPLE: 
Q26 asks about "changements majeurs prédits"
Option C: "la technologie humanisée et l'anticipation 
de l'avenir"

You think: "This is actually a more INTERESTING answer 
than the others. It captures what Kurzweil is really 
about — humanizing technology and anticipating the future."

You are RIGHT that it's interesting.
You are WRONG that it answers the question.

The question doesn't ask what's INTERESTING about Kurzweil.
It asks what CHANGES he predicts.

THE SEDUCTION: The scope trap answer is often more 
INTELLECTUALLY APPEALING than the correct answer.

"Intelligence artificielle et dépollution" sounds BORING.
"Technologie humanisée et anticipation de l'avenir" 
sounds SMART.

But BORING + CORRECT > SMART + WRONG SCOPE.

FIX: You are not grading the QUALITY of the answer.
     You are checking its ACCURACY against the question.
     The most boring, literal answer is often correct.
