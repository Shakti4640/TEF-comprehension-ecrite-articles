🧠 THE PROBLEM: YOU ARE PLAYING AGAINST A DESIGNER

Every chapter so far has taught you to analyze texts and answer questions. But there is a dimension we have only touched indirectly: the exam is not a neutral test. It is a designed challenge, built by a human who deliberately constructs wrong answers to trap you.

THE EXAM DESIGN REALITY:
═══════════════════════════════════════════════════════════

  When an exam writer creates a multiple-choice question,
  they do NOT:
  → Pick 3 random wrong answers
  → Write obviously stupid options
  → Leave the correct answer clearly visible
  
  They DO:
  → Read the text carefully
  → Identify the correct answer
  → Then ENGINEER 3 wrong answers that will ATTRACT 
    students who misunderstood specific aspects
  → Each wrong answer targets a SPECIFIC type of error
  → The wrong answers are designed to be PLAUSIBLE
  
  THIS IS ADVERSARIAL DESIGN.
  The exam writer is your opponent.
  Understanding their strategy gives you an advantage.

🔍 HOW TO APPROACH IT: THE DISTRACTOR ENGINEERING TAXONOMY

After analyzing all 28 answer options across our three texts (correct answers + distractors), a clear pattern emerges. Exam writers use exactly eight techniques to build wrong answers. Every distractor in every exam can be classified into one or more of these eight types.

THE EIGHT DISTRACTOR ENGINEERING TECHNIQUES:
═══════════════════════════════════════════════════════════

  TECHNIQUE 1: WORD BORROWING
  → Takes real words from the text, reassembles them 
    into a statement the text never makes

  TECHNIQUE 2: MEANING INVERSION
  → Takes the text's actual message and FLIPS it 
    to say the opposite

  TECHNIQUE 3: SCOPE SHIFTING
  → Takes a true detail and presents it at the 
    wrong level of generality (Chapter 19 foundation)

  TECHNIQUE 4: HALF-TRUTH CONSTRUCTION
  → Makes one half correct and one half invented,
    banking on the reader latching onto the correct half

  TECHNIQUE 5: OUTSIDE KNOWLEDGE EXPLOITATION
  → Uses information that is TRUE about the topic 
    but NOT stated in this specific text

  TECHNIQUE 6: VOICE MISATTRIBUTION
  → Takes what one person said and presents it as 
    what someone else said (Chapter 21 foundation)

  TECHNIQUE 7: TEMPORAL DISTORTION
  → Presents predictions as facts, future as present,
    or conditional as certain

  TECHNIQUE 8: EMOTIONAL AMPLIFICATION
  → Takes a neutral text element and adds emotional 
    weight that the text does not support

  │                                                  │
  │  THE HALF-TRUTH FORMULA:                         │
  │  ┌─────────────────┬─────────────────┐           │
  │  │  CORRECT HALF   │  INVENTED HALF  │           │
  │  │  "vie éternelle"│  "bonheur de    │           │
  │  │  (in text)      │   tous"         │           │
  │  │                 │  (not in text)  │           │

  │                                                  │

  │  → Your brain sees "vie éternelle" → MATCH!      │

  │  → First-match satisfaction bias                 │
  │  → When you find ONE matching element, your brain│

  │  → They put the correct half FIRST to trigger    │
  │    premature satisfaction                        │

  │  (Chapter 18 Pitfall 3).                         │

✅ WHAT YOU SHOULD DO

DO 1: RUN THE "SAME OR OPPOSITE?" TEST ON EVERY OPTION

THE SAME-OR-OPPOSITE TEST:
═══════════════════════════════════════════════════════════

  For each answer option, find the text passage it 
  relates to, then ask:

  "Does this option say the SAME thing as the text,
   or the OPPOSITE?"
  
  EXAMPLES:
  
  Text: "croissance exponentielle" (growth)
  Option: "s'éteindra" (will die out)
  → OPPOSITE → ELIMINATE
  
  Text: "améliorée" (improved)
  Option: "dépossédera" (will strip)
  → OPPOSITE → ELIMINATE
  
  Text: "fusionné... différences négligeables" (fusion, equality)
  Option: "supérieurs aux hommes" (superior to humans)
  → OPPOSITE → ELIMINATE
  
  Text: "informer" (inform — neutral)
  Option: "dénoncer" (denounce — negative)
  → OPPOSITE TONE → ELIMINATE
  
  SPEED: This test takes 5-10 seconds per option.
  It eliminates roughly 40% of all distractors.

═══════════════════════════════════════════════════════════

DO 2: VERIFY BOTH HALVES OF COMPOUND ANSWERS

THE COMPOUND VERIFICATION PROTOCOL:
═══════════════════════════════════════════════════════════

  When an answer has the form "X et Y":
  
  STEP 1: Check X against the text
  → Is X stated or supported? ✓ or ✗
  
  STEP 2: Check Y against the text
  → Is Y stated or supported? ✓ or ✗
  
  STEP 3: Evaluate
  → Both ✓ → Potentially correct
  → X ✓ + Y ✗ → HALF-TRUTH TRAP → ELIMINATE
  → X ✗ + Y ✓ → HALF-TRUTH TRAP → ELIMINATE
  → Both ✗ → Clearly wrong → ELIMINATE
  
  APPLICATION TO Q26:
  
  A. "vie éternelle" ✓ + "bonheur de tous" ✗ → HALF-TRUTH
  B. "intelligence artificielle" ✓ + "dépollution" ✓ → CORRECT
  C. "technologie humanisée" ⚠️ + "anticipation" ✗ → WEAK
  D. "robotisation quotidien" ✗ + "dictature" ✗ → DOUBLE WRONG

═══════════════════════════════════════════════════════════

DO 3: CHECK FOR DOMAIN-ADJACENT WORD BORROWING

THE DOMAIN-ADJACENT DETECTION METHOD:
═══════════════════════════════════════════════════════════

  QUESTION: "Does this word appear IN the text, or does 
            it just belong to the same DOMAIN?"

  EXAMPLE: Q20 Option A "La grande crevasse"
  
  → "Crevasse" belongs to the ALPINISM domain ✓
  → But does "crevasse" appear IN this text? ✗
  → It's DOMAIN-ADJACENT but NOT TEXT-PRESENT
  → ELIMINATE
  
  EXAMPLE: Q21 Option D "mulets" and "randonnées"
  
  → "Mulets" belongs to the MOUNTAIN TRAVEL domain ✓
  → But does "mulets" appear IN this text? ✗
  → "Randonnées" belongs to the OUTDOOR ACTIVITY domain ✓
  → But does "randonnées" appear IN this text? ✗
  → Both are DOMAIN-ADJACENT but NOT TEXT-PRESENT
  → ELIMINATE
  
  THE RULE:
  Words that SOUND like they belong to the text's topic 
  but DON'T actually appear in the text = MANUFACTURED 
  familiarity. The exam writer chose them BECAUSE they 
  feel right without being right.

═══════════════════════════════════════════════════════════

DO 4: WATCH FOR CONDITIONAL-TO-CERTAIN DISTORTIONS

THE CONDITIONALITY CHECK:
═══════════════════════════════════════════════════════════

  MANY TEXTS USE CONDITIONAL LANGUAGE:
  → "pourrait" (could/might)
  → "serions capables" (we would be capable)
  → "si le rythme continue" (if the pace continues)
  → "il est probable que" (it is probable that)
  
  DISTRACTORS OFTEN REMOVE THE CONDITIONALITY:
  → "pourrait contenir" → "contient" (might contain → contains)
  → "si le rythme continue" → "inéluctable" (if → inevitable)
  → "serions capables" → "seront capables" (would be → will be)
  
  THE DISTORTION:
  UNCERTAIN → CERTAIN
  CONDITIONAL → FACTUAL
  POSSIBLE → DEFINITE
  
  THIS IS TEMPORAL DISTORTION (#7).
  
  DETECTION: When an answer option states something as 
  CERTAIN or DEFINITE, check whether the text uses 
  conditional language for the same idea. If the text 
  says "might" and the answer says "will," it's distorted.

❌ WHAT YOU SHOULD NOT DO

MISTAKE 1: SELECTING AN ANSWER BECAUSE IT USES WORDS FROM THE TEXT

THE WORD BORROWING TRAP IN ACTION:
                                                          
  YOUR THINKING:
  "Option D says 'vedette de la science' and the text 
   says 'vedette de la prospective scientifique' — 
   that's almost the same words! This must be correct!"
  
  REALITY:
  → "Vedette de la science" IS borrowed from the text ✓
  → But "réalisations" is NOT in the text ✗
  → The borrowed words create FALSE FAMILIARITY
  → They make you TRUST the answer without checking 
    every word
  
  THE EXAM WRITER'S CALCULATION:
  → Borrow 3 words from the text to create trust
  → Change 1 word to create the wrong meaning
  → 3 familiar words > 1 unfamiliar word in your brain
  → You select based on the 3, ignoring the 1
  
  DEFENSE: NEVER select an answer just because it 
  contains words from the text. EVERY word must match, 
  not just most of them.

MISTAKE 2: REJECTING AN ANSWER BECAUSE IT DOESN'T USE TEXT WORDS

THE PARAPHRASE REJECTION TRAP:
                                                          
  YOUR THINKING:
  "Option A says 'ne feront plus qu'un' but the text 
   says 'fusionné avec des différences négligeables.' 
   Those aren't the same words, so maybe A is wrong?"
  
  REALITY:
  → "Ne feront plus qu'un" = will become one
  → "Fusionné... différences négligeables" = fused... 
    negligible differences
  → SAME MEANING, different words
  → This is PARAPHRASE, not mismatch
  → The correct answer often REPHRASES the text
  → If it used the EXACT same words, it would be 
    too easy to identify
  
  THE EXAM WRITER'S DESIGN:
  → Correct answer uses PARAPHRASE (different words, same meaning)
  → Distractors use BORROWED WORDS (same words, different meaning)
  → This creates a PARADOX:
    The answer that LOOKS most like the text = often WRONG
    The answer that SOUNDS different from the text = often RIGHT
  
  DEFENSE: Check MEANING match, not WORD match.
  Paraphrase = correct. Word borrowing with distortion = trap.
THE PARADOX VISUALIZED:
═══════════════════════════════════════════════════════════

  TEXT: "êtres humains et machines non biologiques 
        pourraient avoir si parfaitement fusionné que 
        les différences entre eux seraient devenues 
        négligeables"

  OPTION A: "les hommes et les outils technologiques 
             ne feront plus qu'un"
  → Different words → Same meaning → CORRECT ✓
  
  OPTION D: "présenter des réalisations révolutionnaires 
             d'une vedette de la science"
  → Similar words → Different meaning → WRONG ✗
  
  SIMILAR WORDS ≠ CORRECT
  SAME MEANING = CORRECT
  
  This is perhaps THE most important lesson of Chapter 22.

═══════════════════════════════════════════════════════════

MISTAKE 3: CHOOSING THE MOST "IMPRESSIVE" ANSWER

THE SOPHISTICATION TRAP (FULLY EXPLAINED):
                                                          
  THE EXAM WRITER'S PSYCHOLOGY:
  → They know students want to seem smart
  → They know students are drawn to complex-sounding answers
  → So they make one distractor sound MORE INTELLECTUAL 
    than the correct answer
  
  EVIDENCE FROM OUR TEXTS:
  
  Q26: "la technologie humanisée et l'anticipation 
        de l'avenir" (SOUNDS like a TED talk)
  vs.  "l'intelligence artificielle et la dépollution"
       (SOUNDS like a simple list)
  → The simple list = CORRECT
  → The TED talk answer = WRONG
  
  Q27: "ravive l'éternel conflit entre idéalistes 
        et réalistes" (SOUNDS like philosophy class)
  vs.  "renvoie à des questions scientifiques 
        et métaphysiques" (SOUNDS straightforward)
  → The straightforward answer = CORRECT
  → The philosophy answer = WRONG
  
  PATTERN: The correct answer is often the LEAST 
  impressive-sounding option among the plausible ones.
  
  WHY: Because the correct answer MATCHES THE TEXT,
  and the text is a journalistic article, not a 
  philosophy thesis. Simple, accurate reporting 
  produces simple, accurate answers.

MISTAKE 4: NOT RECOGNIZING CROSS-TEXT CONTAMINATION

THE MULTI-TEXT EXAM DANGER:
                                                          
  On a multi-text exam, your brain holds vocabulary 
  from ALL texts simultaneously.
  
  EXAM WRITER EXPLOITATION:
  → They place "patrimoine génétique" (from Texte 102) 
    in a question about Texte 101
  → Your brain recognizes it as "something I just read"
  → You don't check WHICH text it came from
  → You select it because it feels familiar
  
  DEFENSE: When a term feels familiar, ALWAYS ask:
  "Did I see this in THIS text or in ANOTHER text?"
  
  If you can't point to the EXACT LINE in THIS text 
  where the term appears, it may be cross-text 
  contamination.

⚠️ COMMON PITFALLS AND FAILURE MODES

PITFALL 1: THE FIRST-MATCH SATISFACTION TRAP

MECHANISM:
  Your brain scans answer options.
  Option A's first half matches the text.
  DOPAMINE: "Found it!"
  Brain stops scanning.
  You select A without checking the second half.
  The second half is invented.
  WRONG.

THIS IS THE #1 CAUSE OF ERRORS ON COMPOUND ANSWERS.

The exam writer ALWAYS puts the matching half FIRST.
"La vie éternelle (✓) et le bonheur de tous (✗)"
→ Correct bait first, invention second.

DEFENSE: Force yourself to check the SECOND half 
BEFORE confirming your selection. Make it a RULE: 
"I will not select a compound answer until I have 
verified BOTH parts."

PITFALL 2: THE CONFIRMATION BIAS AMPLIFICATION

MECHANISM:
  You have a "gut feeling" about the answer.
  You scan the options.
  One option CONFIRMS your gut feeling.
  You select it with relief.
  
  But your gut feeling was based on GENERAL KNOWLEDGE,
  not on THIS TEXT.
  
  EXAMPLE:
  Gut feeling: "AI will surpass humans"
  Option C: "supérieurs en intelligence aux hommes"
  Gut: "See? That's what I thought!"
  
  But the TEXT says FUSION, not SUPERIORITY.
  Your gut was right about AI in general.
  Your gut was wrong about THIS TEXT.
  
DEFENSE: Treat your gut feeling as a HYPOTHESIS 
to be TESTED, not a conclusion to be confirmed.
If the text SUPPORTS your gut → select.
If the text CONTRADICTS your gut → override.

PITFALL 3: THE TECHNIQUE STACK BLINDSPOT

MECHANISM:
  Some distractors use MULTIPLE techniques simultaneously.
  You detect ONE technique and think "I found the trap!"
  But you miss the SECOND technique.
  
  EXAMPLE: Q26 Option C
  → You detect EMOTIONAL AMPLIFICATION ("humanisée" 
    sounds warmer than "fusionné")
  → You think: "Hmm, the wording is slightly different, 
    but the idea is similar..."
  → You MISS the SCOPE SHIFT ("anticipation" is an 
    activity, not a change)
  → You select C despite detecting one issue
  
  STACKED TECHNIQUES ARE HARDER TO CATCH.
  
DEFENSE: When you find ONE problem with an answer,
don't assume that's the only problem. Check for a 
SECOND issue. Distractors often have multiple flaws 
working together.

PITFALL 4: THE "PROCESS OF ELIMINATION LEFT THE WRONG ONE" SCENARIO

MECHANISM:
  You correctly eliminate 2 options.
  You're left with 2 options.
  One is the correct answer.
  One is a WELL-CONSTRUCTED distractor.
  
  Under time pressure, you pick the distractor because 
  it uses more familiar words from the text.
  
  THE WELL-CONSTRUCTED DISTRACTOR:
  → Borrows heavily from the text (familiar)
  → Sounds sophisticated (impressive)
  → Has ONE hidden distortion (invisible under pressure)
  
  THE CORRECT ANSWER:
  → Paraphrases the text (less familiar)
  → Sounds simpler (less impressive)
  → Has NO distortions (but no familiar word anchors)
  
DEFENSE: When elimination leaves 2 options, apply 
Chapter 18's scoring system. Criterion 1 (Textual Proof) 
and Criterion 3 (Precision) will differentiate them.
The option with MEANING match > the option with WORD match.

🔧 THE COMPLETE DISTRACTOR DETECTION PROTOCOL

THE 4-CHECK DISTRACTOR DETECTION PROTOCOL:
═══════════════════════════════════════════════════════════

  FOR EVERY ANSWER OPTION, RUN THESE 4 CHECKS IN ORDER:

  CHECK 1: SAME OR OPPOSITE? (5 seconds)
  ─────────────────────────────────────
  → Find the text passage this option relates to
  → Does the option say the SAME thing or the OPPOSITE?
  → If OPPOSITE → MEANING INVERSION → ELIMINATE
  → Catches ~40% of distractors
  
  CHECK 2: BOTH HALVES VERIFIED? (10 seconds)
  ─────────────────────────────────────
  → If the option has "X et Y" structure:
  → Is X in the text? Is Y in the text?
  → If one half fails → HALF-TRUTH TRAP → ELIMINATE
  → Catches ~15% of distractors
  
  CHECK 3: TEXT-PRESENT OR DOMAIN-ADJACENT? (5 seconds)
  ─────────────────────────────────────
  → Do the key words ACTUALLY appear in the text?
  → Or do they just SOUND like they belong?
  → If absent from text → WORD BORROWING/FABRICATION → ELIMINATE
  → Catches ~20% of distractors
  
  CHECK 4: CONDITIONAL OR CERTAIN? (5 seconds)
  ─────────────────────────────────────
  → Does the option state something as FACT?
  → Does the text state the same thing as POSSIBLE?
  → If text = conditional and option = certain 
    → TEMPORAL DISTORTION → ELIMINATE
  → Catches ~15% of distractors

  TOTAL TIME: ~25 seconds per option
  TOTAL CATCH RATE: ~90% of all distractors

  THE REMAINING ~10% REQUIRE:
  → Chapter 18 scoring system (for close calls)
  → Chapter 19 scope analysis (for true-but-wrong-scope)
  → Chapter 21 voice tracking (for misattributed claims)
