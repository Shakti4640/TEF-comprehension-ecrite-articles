Texte 101 : Paroles de techno prophète 

Considérés par certains comme un diseur de bonne aventure technologique.
Ray Kurzweil est une vedette de la prospective scientifique. Ses
prédictions ont le don de diviser : cela va du désaccord le plus véhément
à une admiration éperdue frisant le culte du héros en passant par le
scepticisme le plus froid. Il imagine sans ciller un monde futur où,
notamment, de minuscules robots annuleront les effets de la pollution et
où l'humanité "pourra vivre éternellement sans vieillir".

Selon ses estimations, nous serions capables, d'ici à la fin des années
2030, de télécharger le cerveau humain sur un ordinateur, stockant ainsi
«toute la personnalité, la mémoire, les talents et le passé d'une
personne». D'ici là, êtres humains et machines non biologiques pourraient
avoir si parfaitement fusionné que les différences entre eux seraient
devenues négligeables. Dès 2045, l'intelligence humaine, améliorée,
pourrait commencer à s'étendre dans l'univers.

Si le rythme du progrès technologique continue de s'accélérer, comme le
prédit Kurzweil, il est probable que le débat sur la singularité connaîtra
lui aussi une croissance exponentielle. Tant cela fait le lien entre des
thèmes extrêmement controversés (Dieu, la crise énergétique, le génie
génétique) et des concepts de science-fiction (l'intelligence artificielle,
la réalité virtuelle, l'ingénierie moléculaire).

                                        D'après Courrier international

  → UNLESS the change is based on discovering a specific

⚠️ COMMON PITFALLS AND FAILURE MODES

PITFALL 1: THE OVERCONFIDENCE TRAP

SYMPTOM: You answer all 4 questions in 3 minutes 
         because everything "seems obvious."

DANGER: Speed without verification = vulnerability.

  "Obvious" answers are sometimes DESIGNED to feel obvious.
  The half-truth trap (Q26-A) FEELS obvious because 
  "vie éternelle" jumps off the page.
  
  Without the compound check, you select A confidently.
  With the compound check, you catch the invention.
  
  OVERCONFIDENCE MANIFESTS AS:
  → Skipping the compound verification ("both halves look fine")
  → Skipping the scope check ("this obviously answers the question")
  → Skipping the review ("I'm sure about all of them")
  → Finishing 3 minutes early and not using the buffer
  
FIX: Use ALL your allocated time. If you finish early, 
use the remaining time to:
  → Re-read the text passage for each answer
  → Verify each answer against the EXACT text line
  → Check for any compound answers you didn't fully verify
  
  EARLY FINISH + NO VERIFICATION = OVERCONFIDENCE
  EARLY FINISH + THOROUGH VERIFICATION = JUSTIFIED CONFIDENCE

PITFALL 2: THE FATIGUE-INDUCED SHORTCUT

SYMPTOM: By Q27 (the 4th question), you're tired 
         and start cutting corners.

WHAT GETS CUT:
  → Scope dissection ("I'll just read the options")
  → Compound verification ("One half matches, good enough")
  → Framework check ("This sounds right, I'll go with it")
  
WHAT HAPPENS:
  → Q27 is the HARDEST question (scope trap + sophistication)
  → It's also the question where you're MOST tired
  → Maximum difficulty + minimum effort = maximum error risk
  
THE DATA FROM OUR SIMULATION:
  → Q27 took 75 seconds (same as Q26)
  → Q27 required framework substitution detection
  → Q27's most dangerous distractor (B) was the most 
    sophisticated-sounding option
  → Without the framework check, B would have been selected
  
FIX: The efficiency ordering from Chapter 20.
  → Answer EASY questions first (Q24: 50 sec)
  → Answer HARD questions when you still have energy
  → OR: If you're fading, take 5 seconds to breathe 
    before the last question
  → 5 seconds of reset > 30 seconds of fatigued guessing

PITFALL 4: THE CROSS-QUESTION CONTAMINATION

SYMPTOM: Your answer to Q25 influences your answer to Q26, 
         even though they ask different things.

EXAMPLE:
  → Q25: You selected A (fusion — becoming one)
  → Q26: You now think "I already picked the fusion answer, 
    so Q26 must be about something ELSE"
  → You avoid B ("intelligence artificielle et dépollution") 
    because it "sounds like the same topic"
  → You pick C instead
  → WRONG
  
THE REALITY:
  → Q25 asks about the MAIN THEORY (central concept)
  → Q26 asks about SPECIFIC CHANGES (concrete predictions)
  → They are DIFFERENT QUESTIONS about the SAME TEXT
  → The same text elements can answer different questions 
    from different angles
  → "Fusion" is the THEORY (Q25)
  → "AI and depollution" are SPECIFIC CHANGES (Q26)
  → They are complementary, not redundant
  
FIX: RESET your brain between questions.
  → Each question has its OWN scope (Chapter 19)
  → Each question asks about a DIFFERENT aspect
  → Previous answers should NOT influence current ones
  → The text can provide evidence for multiple questions 
    from different angles
  
  TREAT EACH QUESTION AS INDEPENDENT.
  The only connection between questions is the text itself.
