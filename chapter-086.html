Les chauffeurs marquent un point

« Je roulais seulement à 132 km/h… ». On connaît tous les conducteurs sanctionnés par des retraits de points qui crient à l'injustice. Sensibles aux protestations de leurs électeurs, 120 députés ont signé une proposition de loi visant à une « semi-tolérance » pour les infractions minimes au Code de la route. Selon le député Jacques Revelli, les Français approuvent la sévérité des mesures qui ont permis de faire baisser le nombre de morts, mais les petites sanctions seraient vécues comme un harcèlement. QUESTION 24 — What is the goal of the proposed law?

The question: l'objectif poursuivi par cette proposition de loi serait...

Notice: the question itself uses "serait" (conditional) — acknowledging this is a PROPOSAL, not an enacted law.

Step 1: Identify the proposal's stated goal.

"une proposition de loi visant à une « semi-tolérance » pour les infractions minimes au Code de la route"

Key terms:

  • "semi-tolérance" = partial tolerance, partial leniency
  • "infractions minimes" = minor/minimal violations
  • "visant à" = aiming at

The goal: MORE LENIENCY for SMALL violations.

Step 2: Translate this goal into the language of the answer options.

"Semi-tolérance for minor infractions"
= Making the rules LESS STRICT for small violations
= SOFTENING the regulations on minor tickets
= "Assouplir la réglementation sur les contraventions"

Step 3: Test each option.

OptionAnalysisVerdict
A) de modifier le Code de la route actuel"Modifier" (to modify) is too VAGUE. It doesn't specify the DIRECTION of change. Modifying could mean making it stricter OR more lenient. The proposal specifically aims to make it MORE LENIENT for small violations❌ TRAP — technically true but too vague. The exam wants the SPECIFIC direction, not a generic "change"
B) de durcir le système de points du permis"Durcir" = to harden, to make stricter. The proposal aims for "semi-tolérance" = MORE leniency, not MORE strictness. This is the OPPOSITE direction❌ WRONG — direct contradiction. "Semi-tolérance" ≠ "durcir"
C) de diminuer le nombre de contrôles routiersNo reduction in police CONTROLS is mentioned. The proposal changes the SANCTIONS (penalties), not the ENFORCEMENT (controls). Fewer penalties ≠ fewer checks❌ TRAP — confuses changing penalties with reducing enforcement
D) d'assouplir la réglementation sur les contraventions"Assouplir" = to soften, to make more flexible, to relax. "Réglementation sur les contraventions" = regulations on traffic tickets. "Semi-tolérance for minor infractions" = softening the rules on tickets. PERFECT match✅ CORRECT

✅ Answer: D

The "Vague vs. Specific" Trap (Option A)

Option A introduces a trap type that becomes increasingly common in Phases 3-5:

OPTION A: "modifier le Code de la route"
          = change the traffic code

This is TECHNICALLY TRUE — the proposal does aim to change the code.
But it's INSUFFICIENTLY SPECIFIC — it doesn't capture the DIRECTION of change.

OPTION D: "assouplir la réglementation sur les contraventions"
          = soften the regulations on tickets

This is BOTH true AND specific — it captures WHAT changes AND in WHICH direction.

When two options seem true, the MORE SPECIFIC one is correct:

"Modifier" (change)    → could mean stricter OR more lenient → VAGUE
"Assouplir" (soften)   → specifically means more lenient      → SPECIFIC

VAGUE + TRUE < SPECIFIC + TRUE

🔑 Rule: When a question asks about a GOAL or OBJECTIVE, the correct answer specifies the DIRECTION, not just the existence of change. "Modify" is too vague when the text clearly indicates "soften."

The "Penalties vs. Enforcement" Distinction (Option C)

Option C confuses two different components of traffic law:

ENFORCEMENT (contrôles):   How often police CHECK for violations
                          → Number of speed cameras, roadside checks, etc.

PENALTIES (sanctions):     What HAPPENS when a violation is detected
                          → Points deducted, fines issued, etc.

The proposal changes PENALTIES (fewer points for small violations).
It does NOT change ENFORCEMENT (no reduction in police checks).

🔑 Rule: In legal/political texts, distinguish between ENFORCEMENT (how rules are applied) and PENALTIES (what consequences follow violations). Changing one doesn't change the other.

The "Approve...But" Structure — Reading Dual Opinions

This is one of the most important structures in political texts. The French public holds TWO simultaneous opinions:

STRUCTURE:  "Les Français APPROUVENT X... MAIS Y"

OPINION 1:  They approve strictness (for serious violations)
OPINION 2:  They resent small sanctions (for minor violations)

These are NOT contradictory — they apply to DIFFERENT cases:
  Serious cases → strict ✅
  Minor cases → lenient ✅

The exam will try to make you choose ONE opinion and ignore the other:

Option B focuses on OPINION 1 (effectiveness) but REVERSES it → wrong
Option C focuses on OPINION 1 (strictness) but AMPLIFIES it → wrong  
Option D captures OPINION 2 (leniency for certain cases) → correct
Option A ignores BOTH opinions → wrong

🔑 Rule: When a text presents an "approve...but" structure, the PUBLIC OPINION question will target the RESERVATION (the "but" part), not the approval. People's COMPLAINTS are more newsworthy than their satisfaction.

The Conditional "seraient vécues" — Reported Perception

Notice the conditional in "les petites sanctions seraient vécues comme un harcèlement":

"seraient vécues" = would be experienced / are reportedly experienced

This is Deputy Revelli's REPORT of public feeling, not a confirmed fact.
The conditional signals: "this is what people say/feel" (reported perception).

From Chapter 7, you know the conditional marks uncertainty. Here it marks REPORTED PERCEPTION — the deputy is conveying what people feel, filtered through his political interpretation. The exam doesn't challenge this — it treats the deputy's report as the article's source for public opinion. But being aware of this framing makes your reading more precise.

QUESTION 26 — What specifically is proposed?

This question targets the SPECIFIC MECHANISM of the proposed law — not its general goal (Q24) or the public opinion behind it (Q25), but its precise CONTENT.

WHAT is targeted:    Speed violations of less than 5 km/h over the limit
WHAT is proposed:    Don't deduct points UNTIL the THIRD offense
WHAT this means:     First offense → no points deducted
                     Second offense → no points deducted
                     Third offense → points deducted

= The first TWO small violations are PARDONED/FORGIVEN
= Only the THIRD one triggers a penalty

The key construction: "ne...qu'à la troisième infraction"

Remember from Chapter 3: "ne...que" = "seulement" (only)

"ne les sanctionner...QU'À la troisième infraction"
= sanction them ONLY at the third offense
= the first two are forgiven

The "Current vs. Proposed" Trap (Option D)

This is the CENTRAL trap of Chapter 8 — the one the entire chapter was building toward:

OPTION D: "retirer systématiquement des points à tout conducteur fautif"
          = systematically deduct points from every offending driver

THIS IS THE CURRENT SYSTEM:
  → Currently, EVERY speeding violation results in point deduction
  → This is what the proposal wants to CHANGE

THE QUESTION ASKS: "il a été proposé" = what has been PROPOSED
  → The PROPOSED system pardons the first two small violations
  → Option D describes what EXISTS, not what is PROPOSED

CURRENT LAW ≠ PROPOSED CHANGE

This trap works because Option D is a TRUE statement — it accurately describes the existing system. But the question asks about the PROPOSAL, and Option D describes what the proposal aims to REPLACE.

🔑 Rule: In political/legal texts, the exam will offer the CURRENT SITUATION as an answer to a question about the PROPOSED CHANGE (and vice versa). Always check: does the question ask about what IS or what MIGHT BE?

The "Exaggeration" Trap (Option A)

Option A takes the proposal and AMPLIFIES it beyond what the text says:

THE PROPOSAL:   Don't deduct points for the first two small violations
                → Penalties DELAYED, not ELIMINATED

OPTION A:       "Supprimer les contraventions pour les excès de vitesse"
                → Penalties ELIMINATED entirely for ALL speeding

AMPLIFICATION:
  "Delayed until third offense" → "Suppressed entirely"
  "For violations < 5 km/h"    → "For all speeding violations"
  
Both the SCOPE (all speeding vs. small violations only)
and the DEGREE (elimination vs. delay) are exaggerated.

🔑 Rule: Watch for answers that EXAGGERATE the proposal. If the text says "delay penalties for small violations," reject options that say "eliminate penalties for all violations." Check both the SCOPE (which violations?) and the DEGREE (reduce or eliminate?).

⚠️ COMMON PITFALLS AND FAILURE MODES

Pitfall 1: The "Current-as-Proposed" Swap

What happens: The exam offers the CURRENT SITUATION as the answer to a question about the PROPOSED CHANGE.

Full mechanism:

CURRENT SYSTEM:
  "Every speeding violation → automatic point deduction"
  
PROPOSED CHANGE:
  "Small violations → forgiven until third offense"

Q26 asks: "What has been PROPOSED?"

OPTION D answers: "retirer systématiquement des points à tout conducteur fautif"
  → This describes the CURRENT system
  → It's what the proposal wants to CHANGE
  → Answering with the current system is BACKWARDS

The defense: When a question uses "proposé," "envisagé," "prévu," or "visant à," your answer must come from the PROPOSED column, not the CURRENT column. If an option accurately describes the existing situation, it's probably the trap.

Pitfall 2: The "Direction Blindness" Trap

What happens: You focus on the CONTENT of an answer but miss its DIRECTION.

Example: Q24 — comparing options A and D:

Option A: "modifier" le Code    → CHANGE (no direction specified)
Option D: "assouplir" la réglementation → SOFTEN (direction = more lenient)

The text says: "semi-tolérance" → direction = MORE LENIENT

A says "change" without direction → too vague
D says "soften" with clear direction → precise match

The defense: For every option about a PROPOSAL or CHANGE, ask: "In which DIRECTION?"

Stricter?  → "durcir," "renforcer," "aggraver"
More lenient? → "assouplir," "alléger," "adoucir"  
Neutral? → "modifier," "changer," "réviser"

If the text clearly indicates a direction, reject neutral/opposite options.

Pitfall 3: The "Dual Opinion" Confusion

What happens: The text presents a population that holds TWO opinions simultaneously. You select an answer based on only ONE of them.

Example: Q25 — The French approve strictness BUT resent small sanctions.

If you remember ONLY the approval:
  → You might pick C ("souhaitent plus de fermeté") = want MORE strictness
  → WRONG — this ignores their complaint about small sanctions

If you remember ONLY the resentment:
  → You correctly pick D ("réclament plus d'indulgence pour certains cas")
  → CORRECT — this captures the "but" side

If you remember BOTH but can't reconcile them:
  → The "approve...but" structure resolves the tension:
    They approve strictness FOR SERIOUS CASES
    They want leniency FOR MINOR CASES
    = D: "indulgence pour CERTAINS cas" (certain/specific cases)

The key word in Option D is "certains" (certain) — it acknowledges that leniency applies to SOME cases, not ALL. This precision matches the text's dual opinion structure.

Pitfall 4: The Idiomatic Blindness

What happens: You miss the meaning of an idiomatic expression and lose contextual information.

Example: "pied au plancher"

LITERAL:   "foot on the floor" → pressing the gas pedal to the floor
IDIOMATIC: "at full speed," "all out," "full throttle"
CONTEXT:   The journalist uses a CAR metaphor to describe a CAR LOBBY
IRONY:     Using driving language for an organization that defends drivers

This expression tells you the lobby is campaigning AGGRESSIVELY — they're not hesitant or cautious. This helps eliminate Option D ("souhaite le réexamen" = wants re-examination = hesitant/cautious).

🔑 Rule: Idiomatic expressions in journalistic texts often carry TONAL information. Even if you don't know the exact expression, note the ENERGY it conveys (aggressive, cautious, humorous, angry) and use that to evaluate answer options.
